# 美术资源需求清单

> **版本**: v1.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **目标平台**: 微信小游戏 | **主包限制**: < 4MB | **美术风格**: 像素风 / 低面数矢量
> **总资源量预估**: ~2.5MB 主包内, 其余远程加载

---

## 0. 设计前提与约束

### 0.1 微信小游戏平台约束

| 约束 | 数值 | 对美术的影响 |
|------|------|-------------|
| 主包大小 | < 4MB (目标 3.5MB) | 核心资源内置, 非核心远程加载 |
| 启动屏图 | < 100KB | 1 张 PNG, 256 色 |
| 单张图集 | < 512KB | 拆分多个小图集 |
| 内存上限 | < 200MB | 纹理尺寸 ≤ 1024×1024 |
| Draw Call | < 100 | 图集合并, 减少独立 Sprite |
| 帧率 | 60FPS | 序列帧动画帧数 ≤ 8 帧/动作 |
| 音频 | 96kbps, 单文件 < 200KB | 短音效用 ADPCM, BGM 用 MP3 |

### 0.2 美术风格定位

```
风格: 像素风 / 低面数矢量
色调分区:
  遗忘森林: 翠绿 #4A7A4A + 苔藓 #8BAA7A
  幽暗墓穴: 灰紫 #6A5A7A + 鬼火 #AAFFAA
  熔岩火山: 红橙 #AA3A2A + 焦黑 #3A2A1A
  冰封雪原: 蓝白 #7ABBDD + 冰晶 #DDFFFF
  剧毒沼泽: 墨绿 #3A5A2A + 毒紫 #AA77AA
  暗影深渊: 紫黑 #2A1A3A + 虚空 #6644AA
  中性色调: 米白 #F0E8D8 / 深灰 #3A3A3A / 金色 #D4AF37
```

## 10. 命名规范总结

```
统一命名规则: [类型]_[所属]_[名称]_[状态]_[尺寸]

类型前缀:
  bg_       — 背景图
  btn_      — 按钮
  icon_     — 图标 (功能/物品/技能)
  char_     — 角色
  monster_  — 怪物
  miniboss_ — 迷你 Boss
  boss_     — 终结 Boss
  tile_     — 地形 Tile
  fx_       — 特效
  hui_      — UI 组件
  panel_    — 面板
  slider_   — 滑块
  map_      — 地图组件
  card_     — 卡牌
  slot_     — 槽位
  rarity_   — 稀有度
  set_      — 套装
  bgm_      — 背景音乐
  sfx_      — 音效
  font_     — 字体

示例:
  bg_main_menu.png          — 主界面背景
  btn_start_default.png     — 开始按钮(默认态)
  icon_item_healingPotion.png — 回复药水图标
  char_warrior_idle_0001.png  — 战士待机第1帧
  monster_forest_slime_idle.png — 森林史莱姆待机
  tile_forest_floor.png     — 森林区域空地地板
  fx_reaction_burn.png      — 爆燃反应特效
  bgm_combat.mp3            — 战斗BGM
  sfx_ui_click.wav          — UI点击音效
```
