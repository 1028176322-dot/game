## 1. 核心玩法循环 ✅

| 决策项 | 确定方案 | 决策理由 |
|--------|----------|----------|
| 游戏类型 | 回合制 Roguelike 地牢探险 | 适合碎片化场景，肉鸽机制天然保证重玩性 |
| 战斗模式 | **AP 回合制**：每回合 3 行动点，自由分配 | 从"移动→攻击"固定流程改为 AP 分配，提高每回合的决策密度 |
| 棋盘尺寸 | **6×6 网格** | 够小以保持快节奏，够大以支持地形策略 |
| 地牢结构 | **DAG 图**：每层 5~7 房间，2~3 分支汇聚至 Boss | 分支选择 + 汇集感，玩家感觉到"我在朝目标前进" |
| 层数结构 | **3~5 层 + Boss**（初级 3 层，后续扩展至 5 层） | 初期控制内容量；单局 5~15 分钟目标决定最多 5 层 |
| 房间类型 | 6 种：战斗(55%)/宝箱(12%)/回复(10%)/商店(10%)/强化(8%)/**事件(5%)** | **事件房为新增**，提供非战斗决策调味 |

### 已否决方案

| 被否方案 | 否决理由 |
|----------|----------|
| 原设计 "ATK + d6 - DEF" 纯骰子判定 | 玩家决策空间太小，每回合只有"攻击"一个有意义的选择 |
| 每回合固定移动+攻击 | 限制了策略组合的可能性 |
| 无尽模式作为默认 | 内容投入集中在前 3~5 层，无尽模式后续版本再考虑 |

## 2. 战斗系统 ✅（已更新为实时自动攻击 v2.0）

| 决策项 | 确定方案 |
|--------|----------|
| 战斗模式 | **实时自动攻击**：玩家走位控制 + 自动索敌攻击（1次/秒）|
| 操作方式 | 左半屏摇杆移动，右半屏技能按钮，双击翻滚闪避 |
| 地形类型 | 空地 / 障碍物(不可通行) / 荆棘地板(经过扣1) / 治疗符文(站桩回2/秒) / 高台(Range+1,移速-30%) / 暗区(隐身2秒) |
| 怪物 AI | 6 种：冲锋型 / 远程型 / 防御型 / 召唤型(上限3只) / 自爆型(5秒倒计时) / 精英型(混合) |
| 伤害公式 | `伤害 = max(ATK + d6 - DEF×0.5, 1)`；暴击：骰子=6 时 ×1.5 |
| 战斗流程 | 进入房间 → 实时战斗(自动攻击+走位+技能) → 全清结算
| 每回合最多自由分配 AP 子行动 | 支持移动→攻击→再移动→再攻击的切分组合 |

## 3. Build 系统 ✅

| 决策项 | 确定方案 |
|--------|----------|
| 强化层次 | **3 层**：核心能力(A) / 属性强化(B) / 遗物(C) |
| 每层选择 | 每层强化房 3 选 1，每局默认获得 3 次强化（每层 1 次） |
| A-核心能力 (8 种) | 二段斩(普攻AOE) / 穿影(穿人移动) / 怒吼(攻击姿态) / 吸血光环(击杀回血) / 弹射箭(远程弹射) / 盾反(防反) / 时序加速(额外AP) / 防御姿态(原基础) |
| B-属性强化 (7 种) | 均为 **正收益 + 负取舍**（如 ATK+4 但 HP-3，DEF+3 但 Move-1） |
| C-遗物 (6 种) | 荆棘甲 / 幸运币 / 狂战斧 / 不朽石 / 回响之珠 / 暗影斗篷 |
| 有效流派 | 8 种已定义流派（剃刀风暴 / 不动明王 / 暗影刺客 / 狙击手 / 圣骑士 / 时间法师 / 疯狗流 / 地主流）|

## 4. 资源系统 ✅

| 决策项 | 确定方案 |
|--------|----------|
| 资源维度 | **4 种**：HP(生命) / 金币 / 钥匙 / AP(回合内) |
| HP 范围 | 10~50（普通）/ 可通过升级扩展至 80 |
| 金币上限 | 99 |
| 钥匙上限 | 3 |
| AP 上限 | 3（可永久解锁至 4）|
| 受伤代价 | 受伤即损金币 1（HP 归零时金币不清零）|
| 觉悟战 | HP=0 时弹出不屈意志：看广告 50%HP 复活 或 进入觉悟结算 |

## 5. 长线成长（魂石商店） ✅

| 决策项 | 确定方案 |
|--------|----------|
| 可解锁角色 | 4 个：战士(自带盾反·500石) / 弓手(自带弹射箭·500石) / 刺客(自带穿影·800石) / 法师(自带时序加速·1000石) |
| 可解锁天赋 | 3 个：贪婪(魂石+15%·300石) / 探索者(多1房间·400石) / 铁胃(回复翻倍·250石) |
| 可解锁遗物池 | 逐步开放遗物池（每个 600 石）|
| 永久升级 | 初始 AP+1(800石) / 初始携带道具(400石) |
| 魂石结算公式 | 层数×系数 + Boss 击杀奖励(20) |
| 全解锁周期 | 约 80~100 局 [HYPOTHESIS] |

## 6. 广告变现 ✅

| 决策项 | 确定方案 |
|--------|----------|
| 核心原则 | **广告是玩家主动选择，不是关卡设计的补偿** |
| 广告位 1 | 觉悟复活（死亡时 30s 广告 → 50%HP 复活）|
| 广告位 2 | 双倍宝箱（开箱后额外一次）|
| 广告位 3 | 额外强化（强化后多选 1 个）|
| 广告位 4 | 商店打折（单件半价）|
| 广告位 5 | 魂石翻倍（结算界面 ×2）|
| 广告位 6 | 每日宝箱（主界面，5~15 魂石随机）|
| **广告位 7 (新增)** | **跑马灯系统**：每场战斗结束后积累进度，满 3 格领 1 钥匙 |
| 非课金路径 | 所有广告位的奖励均不替代"不观看"的基础体验 |

## 7. 事件房 ✅

| 决策项 | 确定方案 |
|--------|----------|
| 事件房概率 | 5%（约每层 0~1 个）|
| 已设计事件 | 5 种：神秘商人 / 古老石碑 / 毒气室 / 陷阱房 / 幽灵 |
| 事件形式 | 2 选 1 正负权衡决策，无战斗 |

## 8. 新手引导 ✅

| 决策项 | 确定方案 |
|--------|----------|
| 第一局 | 强制教学，怪物 HP 减半，**玩家不可能死亡** |
| 教学顺序 | 移动+攻击 → 回复 → 防御+AP → 强化3选1 → 技能+Boss → 觉悟结算 |
| 第二局起 | 开放全部系统（宝箱/商店/事件/遗物/角色选择）|

## 10. 新增：组合性与随机性系统 (v0.4 追加)

| 新系统 | 层级 | 组合空间 | 核心机制 |
|--------|------|----------|----------|
| 元素反应 | 战斗 | 11 种反应 + 链式反应 | 6 元素两两搭配产生独特效果，链式反应无上限 |
| 词缀系统 | 装备 | 12前缀×12后缀=144种 | 每件装备有唯一名称和玩法暗示 |
| 怪物装配 | 遭遇 | 6×6×12×4×10=17,280种 | 身体+元素+技能+规模+变异 五维组装 |
| 房间变异 | 楼层 | 12种×每层1~2个 | 每层规则不同，影响全局策略 |
| 动态事件 | 事件 | 8场景×6条件×6状态×12后果=3,456+ | 参数化随机生成，玩家状态影响走向 |
| 种子系统 | 全局 | 36^6 ≈ 21.7亿 | 可分享、可复现、可竞速 |

**DDA 动态难度调整**：根据玩家状态自动调整——连续死亡给福利、资源不足给保底、缺引导给提示。

**软锁保护**：商店必有特价药、无钥匙可开木箱、Boss 前篝火、连续死亡 5 次送不朽石。

详见 `design/组合性随机性系统设计.md`
