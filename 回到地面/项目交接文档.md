## 9. 常见问题与注意事项

### 9.1 Cocos Creator 相关问题

| 问题 | 解决方法 |
|------|----------|
| **编辑器打开后显示空场景** | 在 Assets 面板双击 `assets/scenes/splash.scene` 打开场景 |
| **资源显示为紫色/缺失** | 资源 Meta 文件未同步，等编辑器自动导入完成（看右下角进度条） |
| **TypeScript 编译错误** | 编辑器会自动编译，检查 Console 面板的报错信息 |
| **Library 目录缺失** | 编辑器启动后自动生成，无需手动操作 |

### 9.2 微信小游戏相关问题

| 问题 | 解决方法 |
|------|----------|
| **构建后黑屏** | 检查 `build/wechatgame/game.json` 是否包含正确的 AppID |
| **微信开发者工具无法预览** | 检查 `project.config.json` 中的 `appid` 字段 |
| **主包超 4MB** | 检查纹理压缩、删除 unused assets、使用 Atlas |
| **审核被拒** | 参考 `design/美术资源需求清单.md` 附录 B 的审核检查表 |

### 9.3 平台差异

| 注意事项 | 说明 |
|----------|------|
| **路径分隔符** | 代码中统一用 `/`（正斜杠），Windows 路径在引擎内部会自动转换 |
| **文件编码** | 所有文件强制 UTF-8（Windows 默认 GBK，编辑器可能不识别） |
| **类型检查** | macOS/Linux 下 `tsc` 类型检查行为与 Windows 一致 |
| **构建缓存** | 换平台后建议清空 `temp/` 和 `library/` 再构建 |
