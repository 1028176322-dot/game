## 1. 设计目标

**玩家体验**: 主城是玩家在游戏中的所有操作中枢。进入主城即感受到"回家的氛围"——顶部有角色信息、中间有营地背景、底部有功能入口。

**关键规则**:
1. 进入主城时自动加载玩家存档数据
2. 所有核心功能的入口都必须在主城第一屏可见
3. Banner 广告在主城时保持悬挂
4. 跑马灯进度条从 dungeon 移到主城，跨局保存

### 4.2 生命周期

```typescript
onLoad(): void {
    // 加载玩家数据 → 刷新顶部信息
    this._refreshTopBar();

private _bindButtons(): void {

        this.logBtn.node.on(Button.EventType.CLICK, this._onOpenLog, this);
```

### 4.3 顶部信息栏

```typescript
private _refreshTopBar(): void {
    const pdm = PlayerDataManager.getInstance();

    if (this.charNameLabel) {
        this.charNameLabel.string = pdm.getCharacterName() || '冒险者';
    }
    if (this.charClassLabel) {
        const type = pdm.getSelectedCharacterId();
        const names: Record<string, string> = {
            warrior: '小熊·战士', archer: '小鹿·弓手',
            assassin: '小狐·刺客', mage: '小兔·法师', berserker: '小野猪·狂战士',
        };
        this.charClassLabel.string = names[type] ?? '冒险者';
    }
    if (this.levelLabel) {
        this.levelLabel.string = `Lv${pdm.getCharacterLevel()}`;
    }
    if (this.soulStoneLabel) {
        this.soulStoneLabel.string = `魂石: ${pdm.getSoulStones()}`;
    }
    if (this.goldLabel) {
        this.goldLabel.string = `金币: 0`; // 预留
    }
    // 版本号
    if (this.versionLabel) {
        this.versionLabel.string = T('ui.version');
    }
}
```

## 5. 跑马灯管理 (MarqueeManager.ts)

```typescript
/**
 * MarqueeManager - 跑马灯系统
 * 跨局保存进度，主城可见
 */

import { StorageService } from './StorageService';

export class MarqueeManager {
    private static _instance: MarqueeManager | null = null;

    static get instance(): MarqueeManager {
        if (!this._instance) this._instance = new MarqueeManager();
        return this._instance;
    }

    private readonly MAX_SEGMENTS = 3;
    private _segments: number = 0;    // 已点亮格数 (0-3)
    private _keys: number = 0;         // 当前钥匙数

    private constructor() {
        this._load();
    }

    /** 点亮一格 */
    fillSegment(): void {
        if (this._segments < this.MAX_SEGMENTS) {
            this._segments++;
            if (this._segments >= this.MAX_SEGMENTS) {
                this._segments = 0;
                this._keys++;
            }
            this._save();
        }
    }

    /** 消耗钥匙（开宝箱等） */
    useKey(): boolean {
        if (this._keys > 0) {
            this._keys--;
            this._save();
            return true;
        }
        return false;
    }

    /** 看广告点亮一格 */
    watchAdForSegment(): void {
        // 调用 WXAdapter 播放广告
        // 广告成功后调用 fillSegment()
    }

    /** 死亡时重置 */
    resetOnDeath(): void {
        this._segments = 0;
        this._save();
    }

    /** 获取显示文本 */
    getProgressText(): string {
        const filled = '■'.repeat(this._segments);
        const empty = '□'.repeat(this.MAX_SEGMENTS - this._segments);
        return `${filled}${empty}`;
    }

    getKeyCount(): number {
        return this._keys;
    }

    private _load(): void {
        this._segments = StorageService.instance.get('marquee_segments', 0);
        this._keys = StorageService.instance.get('marquee_keys', 0);
    }

    private _save(): void {
        StorageService.instance.set('marquee_segments', this._segments);
        StorageService.instance.set('marquee_keys', this._keys);
    }
}
```

```typescript
private _refreshMarquee(): void {
    const mm = MarqueeManager.instance;
    if (this.marqueeProgress) {
        this.marqueeProgress.string = `跑马灯进度: ${mm.getProgressText()}`;
    }
    if (this.keyCount) {
        this.keyCount.string = `钥匙: x${mm.getKeyCount()}`;

private _onMarqueeAd(): void {
    MarqueeManager.instance.watchAdForSegment();
    // 广告回调后刷新
    this._refreshMarquee();
}
```

## 9. 数据刷新时机

| 时机 | 刷新内容 |
|------|----------|
| `onLoad()` | TopBar 全部 + 跑马灯 |
| 从 dungeon 返回 | 全部刷新（魂石可能变了） |
| 角色选择关闭 | TopBar 角色信息刷新 |
| 商店购买关闭 | TopBar 魂石刷新 |
| 跑马灯广告看完 | 跑马灯进度 + 钥匙数刷新 |
