# Step 8: 战斗结算 / 返回主城 — 详细实现规格

> **版本**: v1.0 | **日期**: 2026-07-01 | **设计者**: GameDesigner  
> **编码**: UTF-8 | **状态**: ⚠️ 死亡结算已有 UI，通关结算需新增，返回主城需修复

## 1. 设计目标

**玩家体验**: 无论通关还是死亡，玩家都获得清晰的结算反馈——杀了多少怪、到了多少层、拿了多少魂石，然后平滑回到主城。

**关键规则**:
1. 死亡 → 觉悟战(复活/放弃) → 结算面板 → 回到主城
2. 通关 → 胜利结算面板 → 回到主城
3. 两种结算共享同一个 "回到主城" 逻辑
4. 魂石、击杀、层数数据跨局保存

---

```
死亡
  │
  └──→ 觉悟战面板

              │
              └──→ 结算面板 (死亡结算)

通关
  │
  └──→ 胜利结算面板
        ├── 显示: 到达区域、层数、击杀、魂石、用时
        ├── [广告: 魂石翻倍]
        └── [回到营地] → 保存数据 → main.scene
```

## 5. GameManager 统一结算接口

```typescript
// GameManager.ts 新增/统一

/** 提交本局结算数据 */
commitRunResult(soulStones: number, floor: number, kills: number): void {
    const pdm = PlayerDataManager.getInstance();

    // 魂石累加到永久存档
    pdm.addSoulStones(soulStones);

    // 更新历史最佳层数
    if (floor > pdm.getBestFloor()) {
        pdm.setBestFloor(floor);
    }

    // 累计击杀数
    pdm.addTotalKills(kills);

    // 总局数 +1
    pdm.addTotalRun();

    // 通关次数（如有）
    // pdm.addVictoryCount();

    pdm.save();

    console.log(`[GameManager] run committed: floor=${floor}, kills=${kills}, stones=${soulStones}`);
}

/** 获取本局用时（秒） */
getRunElapsed(): number {
    // 从 dungeon 场景开始计时到当前
    return this._runElapsed;
}

/** 获取本局魂石数（基于到达层数计算） */
getRunSoulStones(): number {
    // 当前实现：第 1 层 50 魂石，之后每层 +20
    return 50 + (this.currentFloor - 1) * 20;
}
```

## 6. DeathUI.ts 修复详情

### 6.1 当前问题

`onBackToMainClick()` 只发出 `DUNGEON_EXIT` 事件，没有调用 `director.loadScene('main')`。  
`MainSceneController` 没有监听 `DUNGEON_EXIT` 事件，所以玩家卡在地牢场景。

### 6.2 修复代码

```typescript
// DeathUI.ts — onBackToMainClick()

onBackToMainClick(): void {
    const gm = GameManager.instance;
    if (!gm) {
        // 兜底：直接跳转
        director.loadScene('main');
        return;
    }

    // 1. 提交结算数据
    gm.commitRunResult(this._soulStones, this._floorReached, this._kills);

    // 2. 切换游戏阶段
    gm.setPhase(GamePhase.MainMenu);

    // 3. 发出事件（供其他系统清理用）
    eventBus.emit(GameEvent.DUNGEON_EXIT);

    // 4. 关键修复：回到主场景
    console.log('[DeathUI] returning to main city with', this._soulStones, 'soul stones');
    director.loadScene('main');
}
```

## 7. PlayerDataManager 新增方法

```typescript
// PlayerDataManager.ts — 结算相关方法

/** 累加魂石 */
addSoulStones(amount: number): void {
    this._data.soulStones += amount;
}

/** 设置历史最高层 */
setBestFloor(floor: number): void {
    if (floor > this._data.bestFloor) {
        this._data.bestFloor = floor;
    }
}

/** 累加击杀数 */
addTotalKills(kills: number): void {
    this._data.totalKills += kills;
}

/** 总冒险次数 +1 */
addTotalRun(): void {
    this._data.totalRuns++;
}
```
