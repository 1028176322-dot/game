# Step 6: 区域路线选择 — 功能规格

> **版本**: v1.1 | **日期**: 2026-07-01 | **设计者**: GameDesigner  
> **编码**: UTF-8 | **实现方式**: **Panel** — main 场景弹窗

---

## 1. 职责

玩家选择本次冒险的路线（3 区域串联）, 创建 `RunStartConfig`, 由 RunCoordinator 统一启动地牢。

---

## 2. 架构集成

```
MainHub → [开始冒险]
  │
  └── AreaSelectPanel.show()
        │
        ├── 显示默认/可用路线
        ├── 选择路线 → 组装 RunStartConfig
        │
        └── [开始冒险] → RunCoordinator.startRun(config)
              │
              ├── 写入 RunStateService
              └── director.loadScene('dungeon')
```

---

## 3. RunStartConfig

```typescript
// run/RunStartConfig.ts
export interface RunStartConfig {
    characterId: string;      // 当前选中角色
    characterName: string;    // 角色显示名
    zoneRoute: string[];      // 区域路线 ['forest','catacombs','volcano']
    seed: number;             // 随机种子
    difficulty: number;       // 难度系数
    startedAt: number;        // 启动时间戳
    isContinue: boolean;      // 是否继续冒险
}
```

---

## 4. 可用区域与路线

### 区域数据

| ID | 名称 | 难度 | 解锁条件 |
|----|------|------|----------|
| `forest` | 翠绿森林 | 新手 | 默认 |
| `catacombs` | 地下墓穴 | 中等 | 默认 |
| `volcano` | 熔岩火山 | 困难 | 默认 |
| `swamp` | 剧毒沼泽 | 中等 | 通关森林 |
| `tundra` | 永冻冰原 | 困难 | 通关墓穴 |
| `abyss` | 虚空深渊 | 极难 | 通关火山 |

### 路线

| 路线 | 区域串 | 解锁 |
|------|--------|------|
| 新手之路 | 森林→墓穴→火山 | ✅ 默认 |
| 湿地之路 | 森林→沼泽→冰原 | 🔒 通关森林 |
| 深渊之路 | 墓穴→虚空→火山 | 🔒 通关墓穴 |
