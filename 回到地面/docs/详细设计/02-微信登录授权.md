# Step 2: 微信登录授权 — 功能规格

> **版本**: v1.1 | **日期**: 2026-07-01 | **设计者**: GameDesigner  
> **编码**: UTF-8 | **实现方式**: **Panel** — main 场景全屏弹窗

---

## 1. 职责

处理微信小游戏的登录授权。**不再是独立场景**，而是 `main.scene` 下的全屏弹窗。

**触发条件**: `AppFlowController` 进入 `AUTH_CHECK` 状态时，检测到未登录则弹出。

---

## 2. 状态机集成

```
AppFlowController
  │
  ├── AUTH_CHECK
  │     ├── ProfileService.isLoggedIn()
  │     │     ├── true → PROFILE_CHECK
  │     │     └── false → 弹出 LoginPanel
  │     │
  │     ├── LoginPanel.wxLogin() → ProfileService.saveSession()
  │     │     └── 成功 → PROFILE_CHECK
  │     │
  │     └── LoginPanel.guestLogin() → ProfileService.saveGuest()
  │           └── 成功 → PROFILE_CHECK
  │
  └── PROFILE_CHECK
```

---

## 3. 服务贡献

**ProfileService** (`assets/scripts/profile/ProfileService.ts`):

| 方法 | 说明 |
|------|------|
| `wxLogin(): Promise<string>` | 调用 `wx.login()`, 保存 openid |
| `guestLogin(): string` | 生成游客 ID, 保存 |
| `isLoggedIn(): boolean` | 检查本地是否有 openid |
| `getProfile(): UserProfile` | 获取用户资料 |
| `clearLogin(): void` | 清除登录态 |

## 5. 功能点

| # | 功能 | 优先级 |
|---|------|--------|
| 1 | `wx.login()` 获取 code → 本地保存 openid | P0 |
| 2 | 首次登录显示《用户协议》《隐私政策》(可点击) | P0 |
| 3 | 本地保存登录态, 后续启动跳过此 Panel | P0 |
| 4 | 游客模式（无微信环境时可用） | P0 |
| 5 | 登录失败重试（3 次后显示网络异常） | P1 |

## 7. 验收清单

| # | 场景 | 预期 |
|---|------|------|
| 1 | 微信环境 + 未登录 | 弹出 LoginPanel |
| 2 | wxLogin 成功 | ProfileService 保存, 进入 PROFILE_CHECK |
| 3 | wxLogin 失败 | 显示"登录失败", 按钮恢复 |
| 4 | 游客登录 | 生成 guest_id, 进入 PROFILE_CHECK |
| 5 | 浏览器环境 | ProfileService.isLoggedIn() 返回 true, 跳过 Panel |
| 6 | 点击协议文字 | 弹出协议详情弹窗 |
