## 功能点

| # | 功能 | 优先级 |
|---|------|--------|
| 1 | `wx.login()` 获取 code → 换取 openid | P0 |
| 2 | 首次登录显示《用户协议》《隐私政策》弹窗 | P0 |
| 3 | 本地保存 openid + session，后续免登录 | P0 |
| 4 | 游客模式（无微信环境时可用，如浏览器调试） | P0 |
| 5 | 一键登录按钮带微信图标 | P1 |
| 6 | 登录失败重试（3 次后提示网络问题） | P1 |

## 技术实现

```typescript
// PlatformService 新增方法
async function wxLogin(): Promise<{openid: string, sessionKey: string}> {
    const code = await wx.login();
    const result = await request('/api/login', { code });
    StorageService.set('wx_session', result);
    return result;
}
```
