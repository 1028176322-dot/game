## 功能点

| # | 功能 | 优先级 |
|---|------|--------|
| 1 | 输入角色名（6 汉字以内，屏蔽敏感词） | P0 |
| 2 | 5 种动物选择（小熊/小鹿/小狐/小兔/小野猪），点击切换预览 | P0 |
| 3 | 显示当前选择角色的职业名+简介 | P0 |
| 4 | "确认创建"按钮 → 生成 PlayerSaveData | P0 |
| 5 | 角色名随机生成（点击可换） | P1 |
| 6 | 创建动画（动物从地底钻出来） | P2 |
