# Step 3: 创建角色（首次）— 功能规格

> **版本**: v1.1 | **日期**: 2026-07-01 | **设计者**: GameDesigner  
> **编码**: UTF-8 | **实现方式**: **Panel** — main 场景全屏弹窗

---

## 1. 职责

首次启动的玩家创建第一个角色。触发条件: `AppFlowController → PROFILE_CHECK → isFirstTime() === true`。

---

## 2. 状态机集成

```
PROFILE_CHECK
  │
  ├── CharacterService.isFirstTime()
  │     └── true → 弹出 CreatePanel
  │
  ├── CreatePanel
  │     ├── 起名 + 选角 → CharacterService.create()
  │     └── 成功 → MAIN_HUB
  │
  └── MAIN_HUB (非首次)
```

---

## 5. 服务贡献

**CharacterService** (`assets/scripts/profile/CharacterService.ts`):

| 方法 | 说明 |
|------|------|
| `createCharacter(name, typeId): void` | 写入 PlayerSaveData |
| `isFirstTime(): boolean` | 检查是否首次 |
| `getAvailableTypes(): CharacterType[]` | 获取可用角色列表 |
| `getCharacterName(): string` | 获取当前角色名 |
| `selectCharacter(id): void` | 切换当前选中 |
