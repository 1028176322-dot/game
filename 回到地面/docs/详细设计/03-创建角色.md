# Step 3: 创建角色（仅首次）— 详细实现规格

> **版本**: v1.0 | **日期**: 2026-07-01 | **设计者**: GameDesigner  
> **编码**: UTF-8 | **状态**: 🔴 新建

---

## 1. 设计目标

**玩家体验**: 首次进入游戏的玩家起名并选择初始角色，获得归属感和代入感。

**关键规则**:
1. 仅在登录后 `isFirstTime() == true` 时触发
2. 角色名必填，6 汉字以内，屏蔽敏感词
3. 必须选择一个动物角色才能确认
4. 创建成功后写入 `PlayerSaveData`，跳转到主城
5. 5 种动物分别对应 5 个职业，互相独立

## 5. CreateCharacterUI.ts 完整实现

### 5.1 角色数据常量

```typescript
interface CharacterOption {
    id: string;           // warrior / archer / assassin / mage / berserker
    animalName: string;   // 小熊 / 小鹿 / 小狐 / 小兔 / 小野猪
    className: string;    // 战士 / 弓手 / 刺客 / 法师 / 狂战士
    description: string;  // 角色简介
    isDefault: boolean;   // 是否默认解锁

const CHARACTER_OPTIONS: CharacterOption[] = [
    { id: 'warrior',   animalName: '小熊',   className: '战士',   description: '坚实可靠，盾反一切',   isDefault: true },
    { id: 'archer',    animalName: '小鹿',   className: '弓手',   description: '灵动迅捷，弹射制敌',   isDefault: false },
    { id: 'assassin',  animalName: '小狐',   className: '刺客',   description: '穿影无形，一击致命',   isDefault: false },
    { id: 'mage',      animalName: '小兔',   className: '法师',   description: '元素共鸣，毁天灭地',   isDefault: false },
    { id: 'berserker', animalName: '小野猪', className: '狂战士', description: '怒吼冲锋，愈战愈勇',   isDefault: false },
];
```

### 5.3 启动逻辑

```typescript
start(): void {
    // 加载角色卡片
    this._buildAnimalCards();

    // 默认选中战士
    this._selectCharacter('warrior');

    // 绑定按钮
    if (this.confirmBtn) {
        this.confirmBtn.node.on(Button.EventType.CLICK, this._onConfirm, this);
    }

    // 输入框回车事件
    if (this.nameInput) {
        this.nameInput.node.on('editing-did-ended', this._onNameEdited, this);
    }
}

private _buildAnimalCards(): void {
    if (!this.animalCardRoot) return;

    // 清空已有子节点
    this.animalCardRoot.removeAllChildren();

    CHARACTER_OPTIONS.forEach((opt, index) => {
        const card = new Node(opt.id);
        card.setPosition((index - 2) * 120, 0);  // 水平排列，间距 120

        // 卡片背景 (96x112)
        const bg = card.addComponent(Sprite);
        bg.spriteFrame = null; // 运行时加载

        // 动物头像 (64x64) — 子节点
        const avatar = new Node('avatar');
        avatar.setPosition(0, 20);
        // 挂 Sprite，运行时加载对应纹理
        card.addChild(avatar);

        // 动物名标签
        const label = card.addComponent(Label);
        label.string = opt.animalName;
        label.fontSize = 14;
        label.color = new Color(0x33, 0x33, 0x33, 0xFF);
        label.position = new Vec3(0, -30, 0);

        // 点击事件
        card.on(Node.EventType.TOUCH_END, () => {
            this._selectCharacter(opt.id);
        });

        this.animalCardRoot.addChild(card);
        this._animalCards.push(card);
    });
}
```

### 5.4 选择角色

```typescript
private _selectCharacter(id: string): void {
    this._selectedId = id;

    const opt = CHARACTER_OPTIONS.find(c => c.id === id);
    if (!opt) return;

    // 更新按钮文字
    // 首次创建只有 warrior 可选
    if (!opt.isDefault) {
        this._showError(`首次创建只能选择小熊·战士\n其他角色可在商店解锁`);
        return;
    }

    // 更新选中信息
    if (this.selectedInfo) {
        this.selectedInfo.string = `${opt.animalName}·${opt.className}`;
    }
    if (this.selectedDesc) {
        this.selectedDesc.string = `"${opt.description}"`;
    }

    // 高亮选中卡片
    this._animalCards.forEach((card, i) => {
        const bg = card.getComponent(Sprite);
        if (bg) {
            bg.color = CHARACTER_OPTIONS[i].id === id
                ? new Color(0x4A, 0x9E, 0xFF, 0xFF)   // 蓝色选中
                : new Color(0xF0, 0xF0, 0xF0, 0xFF);  // 灰色未选
        }
    });

    this._clearError();
}
```

### 5.5 确认创建

```typescript
private _onConfirm(): void {
    // 1. 验证角色名
    const name = this.nameInput?.string.trim() ?? '';
    if (!name) {
        this._showError('请输入冒险者的名字');
        return;
    }
    if (name.length > 6) {
        this._showError('名字不能超过6个汉字');
        return;
    }
    if (this._hasSensitiveWords(name)) {
        this._showError('名字包含敏感词，请重新输入');
        return;
    }

    // 2. 验证选择
    if (!this._selectedId) {
        this._showError('请选择一个伙伴');
        return;
    }

    // 3. 创建角色
    const pdm = PlayerDataManager.getInstance();
    pdm.createCharacter(name, this._selectedId);

    console.log(`[CreateChar] created: ${name} (${this._selectedId})`);

    // 4. 跳转到主城
    director.loadScene('main');
}

private _hasSensitiveWords(name: string): boolean {
    // TODO: 接入微信敏感词检测 API
    // wx.textToTextCheck(name) 或本地敏感词表
    const blocked = ['admin', 'test', 'root', 'fuck'];
    return blocked.some(w => name.toLowerCase().includes(w));
}
```

### 5.6 辅助方法

```typescript
private _onNameEdited(editBox: EditBox): void {
    const text = editBox.string;
    if (text.length > 6) {
        editBox.string = text.slice(0, 6);
    }
}

private _showError(msg: string): void {
    if (this.errorLabel) {
        this.errorLabel.string = msg;
        this.errorLabel.node.active = true;
    }
    if (this.confirmBtn) {
        this.confirmBtn.interactable = false; // 有错误时禁用按钮
    }
}

private _clearError(): void {
    if (this.errorLabel) {
        this.errorLabel.node.active = false;
    }
    if (this.confirmBtn) {
        this.confirmBtn.interactable = true;
    }
}
```

## 6. PlayerDataManager.ts 改动

```typescript
// 新增方法

/** 创建角色：初始化永久存档 */
createCharacter(name: string, characterType: string): void {
    this._data = {
        soulStones: 0,
        unlockedCharacters: ['warrior'],      // 默认解锁战士
        selectedCharacter: characterType,     // 当前选中
        selectedTalent: null,
        unlockedRelicPoolExtras: [],
        bestFloor: 0,
        totalKills: 0,
        totalRuns: 0,
        version: 1,
    };

    // 添加自定义数据
    this._data['characterName'] = name;
    this._data['createdAt'] = Date.now();
    this._data['totalPlayTime'] = 0;

    this.save(); // 写入本地存储
    console.log('[PlayerData] character created:', name, characterType);
}
```

## 8. 验证规则

| 规则 | 条件 | 反馈 |
|------|------|------|
| 名称为空 | `name.trim() === ''` | 显示"请输入冒险者的名字"，禁止创建 |
| 名称超长 | `name.length > 6` | 输入框自动截断到 6 字 |
| 敏感词 | `hasSensitiveWords(name)` | 显示"名字包含敏感词" |
| 未选择角色 | `_selectedId === null` | 显示"请选择一个伙伴" |
| 选择非默认角色 | `isDefault === false` | 显示"首次只能选择小熊·战士" |
| 输入完成+已选 | 全部通过 | 按钮可点击，创建后跳转 |
