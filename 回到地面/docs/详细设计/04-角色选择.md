## 功能点

| # | 功能 | 优先级 |
|---|------|--------|
| 1 | 显示已有角色列表（角色名/等级/最高层/总次数） | P0 |
| 2 | 点击角色 → 显示详细面板 + "选择"按钮 | P0 |
| 3 | 未解锁角色显示所需魂石 + "解锁"按钮 | P0 |
| 4 | "进入地牢" → 跳转区域选择 | P0 |
| 5 | "管理角色" → 重命名/删除角色 | P1 |
| 6 | 首次创建的角色自动成为唯一角色 | P0 |
| 7 | 8 个角色槽位上限 | P1 |
