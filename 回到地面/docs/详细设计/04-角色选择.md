# Step 4: 角色选择（非首次）— 功能规格

> **版本**: v1.1 | **日期**: 2026-07-01 | **设计者**: GameDesigner  
> **编码**: UTF-8 | **实现方式**: **Panel** — main 场景弹窗

---

## 1. 职责

在 MainHub 点击"角色"时弹出, 显示所有已创建/可解锁的角色列表。

---

## 2. UI 布局 (500x600 弹窗)

```
CharacterPanel (全屏遮罩)
├── Panel (500x600, 居中)
│   ├── TitleLabel (Label, "选择冒险者")
│   ├── SoulStoneLabel (Label, "魂石: {count}", 右上)
│   │
│   ├── CurrentCard (Node, 440x100)       ← 当前选中角色
│   │   ├── Avatar (Sprite, 64x64)
│   │   ├── CurrentName (Label, "小熊·战士")
│   │   ├── CurrentInfo (Label, "小勇 | 最高层: 5")
│   │   ├── CurrentStats (Label, "总冒险: 12 次")
│   │   ├── [进入地牢] (Button)
│   │   └── [管理角色] (Button)
│   │
│   ├── SectionLabel (Label, "── 其他冒险者 ──")
│   ├── SlotContainer (ScrollView, 460x360)
│   │   ├── Card_1 (小鹿·弓手)  → [选择] / [创建] / [解锁: 800魂石]
│   │   ├── Card_2 (小狐·刺客)  → [解锁: 800魂石]
│   │   ├── Card_3 (小兔·法师)  → [解锁: 1000魂石]
│   │   └── Card_4 (小野猪·狂战士) → [解锁: 1200魂石]
│   │
│   └── CloseBtn (Label, "关闭")
```

---

## 3. 操作逻辑

| 操作 | 服务调用 | 说明 |
|------|----------|------|
| [选择] | `CharacterService.selectCharacter(id)` | 切换当前角色 |
| [创建] | `CharacterService.createCharacter(name, id)` | 已解锁类型创建角色 |
| [解锁] | `CharacterService.unlockCharacter(id)` | 消耗魂石解锁 |
| [进入地牢] | 关闭 Panel → 打开 AreaSelectPanel | 开始新冒险 |
| [关闭] | `UiRouter.close('character')` | 返回 MainHub |
