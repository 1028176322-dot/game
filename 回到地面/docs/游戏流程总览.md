## 流程总览

```
                    ┌──────────────────┐
                    │    Splash 启动    │
                    │   (Logo + 加载)   │
                    └────────┬─────────┘
                             │
                    ┌────────▼─────────┐
                    │   微信登录授权     │
                    │  (wx.login + 弹窗) │
                    └────────┬─────────┘
                             │ 首次
                    ┌────────▼─────────┐
                    │  创建角色(仅首次)  │
                    │   起名 + 选动物    │
                    └────────┬─────────┘
                             │
                    ┌────────▼─────────┐
                    │  角色选择(非首次) │
                    │  展示已有角色列表  │
                    └────────┬─────────┘
                             │
                    ┌────────▼─────────┐
                    │     主城界面      │
                    │  核心交互中枢      │
                    ├──────────────────┤
                    │  ① 开始冒险       │
                    │  ② 角色/编队      │
                    │  ③ 商店           │
                    │  ④ 设置           │
                    │  ⑤ 冒险日志       │
                    └────────┬─────────┘
                             │ 点击"开始冒险"
                    ┌────────▼─────────┐
                    │ 区域选择/路线选择  │
                    │  DAG 地图预览     │
                    └────────┬─────────┘
                             │
                    ┌────────▼─────────┐
                    │     地牢战斗      │
                    │  (现有 dungeon )  │
                    ├──────────────────┤
                    │ 胜利 → 跑马灯 →   │
                    │ 下一房间/下一区域  │
                    │ 死亡 → 觉悟战 →   │
                    │ 复活/结算         │
                    │ 通关 → 胜利结算   │
                    └────────┬─────────┘
                             │ 结算完成
                    ┌────────▼─────────┐
                    │   返回主城界面     │
                    │   (循环)          │
                    └──────────────────┘
```

## 场景切换图

```
splash
  │ 加载完成
  ├── 未登录 → login
  │                │ 登录成功
  │                ├── 首次 → create_character
  │                │                     │ 创建完成
  │                │                     └──→ main
  │                └── 非首次 → main
  └── 已登录
       ├── 首次 → create_character → main
       └── 非首次 → main

main
  ├── [开始冒险] → area_select (弹窗/场景) → dungeon
  ├── [角色]     → character_select (弹窗)
  ├── [商店]     → shop (弹窗, 现有)
  ├── [设置]     → settings (弹窗)
  └── [冒险日志] → adventure_log (弹窗)

dungeon
  ├── 死亡 → 觉悟战
  │         ├── [复活] → 继续 dungeon
  │         └── [结算] → settlement → main
  ├── 通关 → victory_settlement → main
  └── 返回 → main
```
