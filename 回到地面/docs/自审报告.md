## 三、广告合规自查

### 3.1 广告类型与位置

| 广告位 | 类型 | 触发时机 | 合规说明 |
|--------|------|----------|----------|
| 觉悟复活 | 激励视频 | 玩家死亡时手动点击 | 用户主动触发，可跳过 |
| 魂石翻倍 | 激励视频 | 结算时手动点击 | 用户主动触发，可跳过 |
| 跑马灯 | 激励视频 | 战斗结束后手动点击 | 用户主动点击点亮 |
| 宝箱 | 激励视频 | 宝箱房手动点击 | 用户主动触发 |
| 插屏 | 插屏广告 | 死亡结算后展示 | 不打断游戏流程 |
| Banner | Banner | 主界面常驻 | 战斗时自动隐藏 |

### 3.2 广告合规检查

| 检查项 | 结果 | 说明 |
|--------|:----:|------|
| 所有广告位是否用户主动触发 | ✅ 是 | 激励广告全部手动点击，插屏在非战斗时展示 |
| 广告是否遮挡游戏核心内容 | ✅ 否 | Banner仅在主界面显示 |
| 广告是否干扰正常游戏操作 | ✅ 否 | 战斗时Banner/Banner已隐藏 |
| 是否有误导性广告关闭按钮 | ✅ 否 | 使用微信标准广告组件 |
| 广告加载失败是否有fallback | ✅ 是 | 直接发放奖励，不阻塞玩家 |
| 是否提供个性化广告关闭选项 | ✅ 是 | 设置中可关闭个性化推荐 |
