# 项目架构根因治理执行方案

> 项目：回到地面  
> 引擎：Cocos Creator 3.8.8  
> 平台：微信小游戏  
> 目标读者：后续执行改造的 AI / 开发者  
> 原则：优先从根因解决问题，允许大改代码；不以调阈值、补丁式兜底作为主要方案。

---

## 0. 执行边界

本方案不是普通优化建议，而是面向落地执行的重构计划。执行时请遵守：

1. 每个阶段独立提交，不能一次性大面积改完。
2. 每个阶段必须能运行或至少能 `tsc` 类型检查通过。
3. 先搭基础设施，再迁移业务代码。
4. 不删除用户现有资源和配置，先建立新管线和门禁。
5. 旧系统保留兼容入口，迁移完成后再清理。

推荐执行顺序：

```text
Phase 1  配置系统根治
Phase 2  随机数/种子系统根治
Phase 3  事件总线类型化和作用域拆分
Phase 4  DungeonSceneController 拆分
Phase 5  资源加载和微信分包
Phase 6  存档/平台服务统一
Phase 7  战斗实体和怪物 AI 拆分
Phase 8  UI prefab/ViewModel 化
Phase 9  工程化门禁
```

## 1. 当前关键问题

### 1.1 配置 JSON 没有成为权威数据源

问题文件：

```text
E:/game/回到地面/assets/scripts/core/ConfigManager.ts
```

现状：

- `loadAll()` 使用内置默认配置。
- `_tryLoadJsonAsync()` 是空实现。
- `assets/resources/config/*.json` 存在，但很多运行逻辑不真正依赖它。

根因：

配置系统没有异步启动流程。业务代码在配置尚未加载前就开始运行，于是只能依赖内置默认值。

目标：

```text
配置表 JSON = 运行时唯一权威数据
TypeScript 默认值 = 开发兜底，不参与正式运行
启动流程 = 等待 ConfigService.loadAll() 完成后再进主界面/地牢
```

### 2.4 `ConfigService.ts` 核心代码

替代 `ConfigManager` 的主加载逻辑。旧 `ConfigManager` 暂时改成代理，避免一次性迁移全部调用。

```ts
import { resources, JsonAsset } from 'cc';
import { GameConfigs, MonstersData, ZonesData } from './ConfigTypes';

type ConfigName = keyof GameConfigs;

const CONFIG_NAMES: ConfigName[] = [
  'battle',
  'economy',
  'elements',
  'equipment',
  'items',
  'monsters',
  'player',
  'skills',
  'text',
  'zones',
];

export class ConfigService {
  private static _instance: ConfigService | null = null;
  private _configs: Partial<GameConfigs> = {};
  private _loaded = false;

  static get instance(): ConfigService {
    if (!this._instance) this._instance = new ConfigService();
    return this._instance;
  }

  get loaded(): boolean {
    return this._loaded;
  }

  async loadAll(): Promise<void> {
    const results = await Promise.all(CONFIG_NAMES.map((name) => this._loadOne(name)));
    for (const [name, data] of results) {
      (this._configs as any)[name] = data;
    }
    this._validateCrossReferences();
    this._loaded = true;
  }

  private _loadOne<T extends ConfigName>(name: T): Promise<[T, GameConfigs[T]]> {
    return new Promise((resolve, reject) => {
      resources.load(`config/${name}`, JsonAsset, (err, asset) => {
        if (err || !asset) {
          reject(new Error(`config/${name}.json load failed: ${err?.message ?? 'unknown'}`));
          return;
        }

        const raw = asset.json as any;
        const data = raw.data ?? this._normalizeLegacyConfig(name, raw);
        this._validateOne(name, data);
        resolve([name, data as GameConfigs[T]]);
      });
    });
  }

  private _normalizeLegacyConfig(name: ConfigName, raw: any): unknown {
    // 兼容现有配置。执行阶段按实际 JSON 结构补全。
    if (name === 'zones') {
      return {
        zonePool: raw.zonePool,
        runsPerGame: raw.runsPerGame,
        zones: raw.zones,
        roomTypeWeights: raw.roomTypeWeights,
      };
    }
    if (name === 'monsters') {
      const { metadata, monsterScale, ...zones } = raw;
      const monsters: Record<string, any> = {};
      for (const zoneId of Object.keys(zones)) {
        for (const monsterId of Object.keys(zones[zoneId])) {
          monsters[monsterId] = { id: monsterId, zone: zoneId, ...zones[zoneId][monsterId] };
        }
      }
      return { monsters, monsterScale };
    }
    return raw;
  }

  private _validateOne(name: ConfigName, data: unknown): void {
    if (!data || typeof data !== 'object') {
      throw new Error(`${name}.json must be an object`);
    }
  }

  private _validateCrossReferences(): void {
    const zones = this.zones;
    const monsters = this.monsters;

    for (const zoneId of zones.zonePool) {
      const zone = zones.zones[zoneId];
      if (!zone) throw new Error(`zones.zonePool references missing zone: ${zoneId}`);
      for (const monsterId of zone.monsterPool) {
        if (!monsters.monsters[monsterId]) {
          throw new Error(`zone ${zoneId} monsterPool references missing monster: ${monsterId}`);
        }
      }
    }
  }

  get zones(): ZonesData {
    return this._configs.zones as ZonesData;
  }

  get monsters(): MonstersData {
    return this._configs.monsters as MonstersData;
  }

  get<T extends ConfigName>(name: T): GameConfigs[T] {
    const cfg = this._configs[name];
    if (!cfg) throw new Error(`Config not loaded: ${name}`);
    return cfg as GameConfigs[T];
  }
}
```

### 3.3 `Rng.ts` 核心代码

```ts
export class Rng {
  private _state: number;

  constructor(seed: number) {
    this._state = seed >>> 0;
    if (this._state === 0) this._state = 0x12345678;
  }

  next(): number {
    // xorshift32
    let x = this._state;
    x ^= x << 13;
    x ^= x >>> 17;
    x ^= x << 5;
    this._state = x >>> 0;
    return this._state / 0xffffffff;
  }

  int(min: number, max: number): number {
    return Math.floor(this.next() * (max - min + 1)) + min;
  }

  chance(probability: number): boolean {
    return this.next() < probability;
  }

  pick<T>(items: T[]): T {
    if (items.length === 0) throw new Error('Rng.pick empty array');
    return items[this.int(0, items.length - 1)];
  }

  weighted<T>(items: T[], getWeight: (item: T) => number): T {
    const total = items.reduce((sum, item) => sum + Math.max(0, getWeight(item)), 0);
    if (total <= 0) return this.pick(items);
    let roll = this.next() * total;
    for (const item of items) {
      roll -= Math.max(0, getWeight(item));
      if (roll <= 0) return item;
    }
    return items[items.length - 1];
  }

  fork(label: string): Rng {
    let hash = this._state;
    for (let i = 0; i < label.length; i++) {
      hash = ((hash << 5) - hash + label.charCodeAt(i)) >>> 0;
    }
    return new Rng(hash);
  }

  get state(): number {
    return this._state;
  }
}
```

### 4.4 `TypedEventBus.ts`

```ts
type Handler<T> = (payload: T) => void;

interface Entry<T> {
  handler: Handler<T>;
  target?: object;
  once: boolean;
}

export class TypedEventBus<EventMap extends Record<string, any>> {
  private _events = new Map<keyof EventMap, Set<Entry<any>>>();

  on<K extends keyof EventMap>(event: K, handler: Handler<EventMap[K]>, target?: object): void {
    if (!this._events.has(event)) this._events.set(event, new Set());
    this._events.get(event)!.add({ handler, target, once: false });
  }

  once<K extends keyof EventMap>(event: K, handler: Handler<EventMap[K]>, target?: object): void {
    if (!this._events.has(event)) this._events.set(event, new Set());
    this._events.get(event)!.add({ handler, target, once: true });
  }

  emit<K extends keyof EventMap>(event: K, payload: EventMap[K]): void {
    const entries = this._events.get(event);
    if (!entries) return;
    const remove: Entry<any>[] = [];
    for (const entry of entries) {
      entry.handler(payload);
      if (entry.once) remove.push(entry);
    }
    for (const entry of remove) entries.delete(entry);
  }

  offTarget(target: object): void {
    for (const [event, entries] of this._events) {
      for (const entry of [...entries]) {
        if (entry.target === target) entries.delete(entry);
      }
      if (entries.size === 0) this._events.delete(event);
    }
  }
}
```

```ts
import { EquipmentSystem } from '../battle/EquipmentSystem';
import { ItemSystem } from '../battle/ItemSystem';

export type ClearedRoomType = 'normal' | 'elite' | 'boss';

export class RewardService {
  constructor(
    private readonly equipmentSystem: EquipmentSystem | null,
    private readonly itemSystem: ItemSystem | null,
  ) {}

  grantRoomClearRewards(roomType: ClearedRoomType): void {
    this._dropEquipment(roomType);
    this._dropItems(roomType);
  }

  private _dropEquipment(roomType: ClearedRoomType): void {
    if (!this.equipmentSystem) return;
    const drops = this.equipmentSystem.generateDrops(roomType, 1);
    for (const drop of drops) {
      if (drop) this.equipmentSystem.pickupToBackpack(drop);
    }
  }

  private _dropItems(roomType: ClearedRoomType): void {
    this.itemSystem?.tryDrop(roomType);
  }
}
```

### 5.5 `RoomFlowController.ts`

```ts
import { RoomType } from '../core/Constants';
import { GameManager, GameEvent } from '../core/GameManager';
import { DungeonManager } from '../dungeon/DungeonManager';
import { RewardService } from './RewardService';
import { eventBus } from '../core/EventBus';

export class RoomFlowController {
  constructor(
    private readonly dungeonManager: DungeonManager,
    private readonly rewardService: RewardService,
  ) {}

  onBattleVictory(): void {
    const room = this.dungeonManager.currentRoom;
    if (!room) return;

    const roomType = room.type === RoomType.Boss
      ? 'boss'
      : room.type === RoomType.Elite
        ? 'elite'
        : 'normal';

    this.rewardService.grantRoomClearRewards(roomType);

    if (room.type === RoomType.Boss) {
      this._handleBossVictory();
    }
  }

  private _handleBossVictory(): void {
    const gm = GameManager.instance;

    if (this.dungeonManager.floorState?.isMiniBossFloor) {
      if (gm.advanceStage()) {
        this.dungeonManager.resetForZone(gm.currentZone, gm.currentStageId);
      } else {
        this.dungeonManager.enterNextFloor();
      }
      return;
    }

    if (gm.isLastStageInZone) {
      eventBus.emit(GameEvent.ZONE_BOSS_DEFEATED, gm.currentZone);
    }
  }
}
```

```ts
  onLoad(): void {
    this._wireSystems();
    this._wireServices();
    this._wireEvents();
  }

  private _wireSystems(): void {
    if (!this.player || !this.gridManager || !this.battleManager || !this.dungeonManager) {
      throw new Error('DungeonSceneController missing required scene bindings');
    }
    this.player.init(this.gridManager);
    this.battleManager.init(this.player, this.gridManager);
    this.dungeonManager.init(this.player, RunRng.instance.seed);
  }

  private _wireServices(): void {
    const reward = new RewardService(this.equipmentSystem, this.itemSystem);
    this._roomFlow = new RoomFlowController(this.dungeonManager!, reward);
  }

  private _wireEvents(): void {
    eventBus.on('battle:victory', () => this._roomFlow?.onBattleVictory(), this);
  }

  onDestroy(): void {
    eventBus.offTarget(this);
  }
}
```

### 5.7 验收标准

3. 区域推进逻辑不再出现在场景控制器。

```ts
import { assetManager, Asset, AssetManager } from 'cc';

export class AssetBundleService {
  private static _instance: AssetBundleService | null = null;
  private _bundles = new Map<string, AssetManager.Bundle>();

  static get instance(): AssetBundleService {
    if (!this._instance) this._instance = new AssetBundleService();
    return this._instance;
  }

  async loadBundle(name: string): Promise<AssetManager.Bundle> {
    const cached = this._bundles.get(name);
    if (cached) return cached;

    const bundle = await new Promise<AssetManager.Bundle>((resolve, reject) => {
      assetManager.loadBundle(name, (err, loaded) => {
        if (err || !loaded) reject(err ?? new Error(`loadBundle ${name} failed`));
        else resolve(loaded);
      });
    });
    this._bundles.set(name, bundle);
    return bundle;
  }

  async load<T extends Asset>(bundleName: string, path: string, type: new (...args: any[]) => T): Promise<T> {
    const bundle = await this.loadBundle(bundleName);
    return new Promise<T>((resolve, reject) => {
      bundle.load(path, type, (err, asset) => {
        if (err || !asset) reject(err ?? new Error(`load asset failed: ${bundleName}:${path}`));
        else resolve(asset as T);
      });
    });
  }

  releaseBundle(name: string): void {
    const bundle = this._bundles.get(name);
    if (!bundle) return;
    bundle.releaseAll();
    assetManager.removeBundle(bundle);
    this._bundles.delete(name);
  }
}
```

### 7.3 StorageService 核心代码

```ts
import { sys } from 'cc';

export class StorageService {
  private static _instance: StorageService | null = null;
  private _isWX = sys.platform === 'wechatgame' || typeof wx !== 'undefined';

  static get instance(): StorageService {
    if (!this._instance) this._instance = new StorageService();
    return this._instance;
  }

  getJson<T>(key: string, fallback: T): T {
    try {
      const raw = this._isWX ? wx.getStorageSync(key) : localStorage.getItem(key);
      if (!raw) return fallback;
      return JSON.parse(raw) as T;
    } catch (err) {
      console.warn(`[StorageService] read failed: ${key}`, err);
      return fallback;
    }
  }

  setJson<T>(key: string, value: T): boolean {
    try {
      const raw = JSON.stringify(value);
      if (raw.length > 1024 * 200) {
        console.warn(`[StorageService] large value: ${key}, ${raw.length} bytes`);
      }
      if (this._isWX) wx.setStorageSync(key, raw);
      else localStorage.setItem(key, raw);
      return true;
    } catch (err) {
      console.warn(`[StorageService] write failed: ${key}`, err);
      return false;
    }
  }

  remove(key: string): void {
    try {
      if (this._isWX) wx.removeStorageSync(key);
      else localStorage.removeItem(key);
    } catch {
      // ignore
    }
  }
}
```
