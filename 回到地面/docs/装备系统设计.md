# 装备系统 · 完整设计

> **版本**: v2.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **内容**: 8 槽位装备 + 24 词缀 + 套装系统 + 附魔 + 背包

---

## 0. 设计总则

```
装备回到地面中的定位:
  ① 每局内收集, 跨局重置 (无装备继承)
  ② 提供数值增量, 但不决定流派走向
  ③ 词缀让每件装备有"名字" (而非"铁剑+3")
  ④ 橙色套装创造收集驱动——凑齐 2/6/8 件触发不同效果
```

### 0.1 装备对战力贡献比

```
装备对玩家最终战斗力的贡献约 15~25%:
  ① 核心能力 + 属性强化: 60%
  ② 装备 (含套装效果):  25%
  ③ 遗物 + 主动技能:   15%
```

## 3. 套装系统 (橙色传奇装备)

### 3.1 套装机制

```
橙色装备 = 传奇品质 + 套装标签
每件橙色装备上标记所属套装名 (如"狂风之盔", "铁壁胸甲")

集齐同一套装的件数触发套装效果:
  2 件: 基础套装效果 (提升对应流派的核心能力)
  6 件: 进阶套装效果 (大幅增强流派战斗节奏)
  8 件: 终极套装效果 (质变, 改变玩法规则)
```

## 4. 套装强度对比与平衡

### 4.1 各套装理论 DPS 提升

```
假设: 玩家基础 ATK=8, 攻击间隔 1.0s, 无套装

狂风 8件:  攻速 +12% → DPS +12% + 风刃(≈15%) = 总 DPS 提升约 30%
铁壁 8件:  护盾常驻 → 有效 HP 提升约 60% (不是 DPS, 是生存)
暗影 8件:  击杀隐身→双倍 → 清图效率提升约 40% (单体 Boss 战提升约 15%)
狂怒 8件:  低血量 +ATK+攻速+暴击 → 残血 DPS 提升约 80% (但风险高)
霜噬 8件:  对被控敌人 +25% + 冻结 → 对单体 Boss 提升约 30%
圣光 8件:  假死+持续回血 → 生存提升约 50% (不是 DPS)

平衡原则: 输出越高的套装风险越大 (狂怒=高风险高回报)
          生存越高的套装输出越低 (铁壁/圣光=稳但慢)
```

### 4.2 套装掉落规则

```
橙色装备 (套装部件):
  掉落来源: Boss 房必掉 1 件 / 金箱(需钥匙)低概率 / 精英战斗房低概率(5%)
  掉落规则: 随机从 6 个套装中选 1 个 → 随机从 8 个槽位中选 1 个
  
  同一种子: 不会掉落已装备的同名同槽位装备
  (防止: 你已经装备了"狂风头盔", 又掉一个"狂风头盔"→ 无用重复)
  
  单局限制: 最多获得 6 件橙色装备 (凑不齐 8 件很正常)
  设计意图: 凑齐 8 件 = 非常幸运, 不是每局都能达到
```

### 4.3 套装 + 词缀 = 完整装备名

```
橙色装备 = [前缀] + [套装名] + [装备槽位] + [后缀]

例:
  锋利的狂风之剑之火焰
  = 锋利(Sharp, +2ATK) + 狂风(攻速+12%) + 剑(武器) + 之火焰(火属性)

  坚固的铁壁护手之荆棘
  = 坚固(Sturdy, +2DEF) + 铁壁(受到伤害-8%) + 护手 + 之荆棘(反伤2点)

  迅捷的暗影之靴
  = 迅捷(Swift, 移速+5%) + 暗影(移速+15%) + 鞋子

每个橙色装备 = 1 个前缀 + 1 个后缀 + 1 个套装标签 = 3 重效果叠加
```

## 5. 背包与装备管理 (更新版)

### 5.1 背包容量

| 项目 | 上限 | 说明 |
|------|------|------|
| 已装备 | **8 件** | 武器+戒指+项链+头盔+胸甲+裤子+鞋子+护手 |
| 背包库存 | **12 件** | 备用装备, 可随时替换 |
| 道具 | 5 个 | 药水/钥匙/临时Buff |

### 5.2 装备替换

```
玩家可在背包中随时替换任意槽位的装备 (战斗中也可, 无 CD)。

例:
  打到"狂风胸甲" → 打开背包 → 选中胸甲槽 → 替换 → 穿上
  检查套装计数: 2/8 件狂风 → 触发攻速+12%!
```

### 5.3 套装计数器 UI

```
装备界面右上角显示套装计数器:

  ┌─────────────────────────┐
  │  套装进度               │
  │                         │
  │  狂风 [■■□□□□□□] 2/8   │  ← 攻速+12% 已激活
  │  铁壁 [□□□□□□□□] 0/8   │
  │  暗影 [■■■■■■□□] 6/8   │  ← 翻滚增伤已激活!
  │  狂怒 [□□□□□□□□] 0/8   │
  │  霜噬 [□ □□□□□□□□] 1/8  │
  │  圣光 [□□□□□□□□] 0/8   │
  └─────────────────────────┘

混合套装策略:
  玩家可以混合搭配不同套装的部件:
  例: 狂风 2 件 (攻速) + 暗影 2 件 (移速) + 铁壁 2 件 (减伤)
```

## 6. 元素附魔 (同 v1.0)

### 6.1 附魔总表

| 附魔 | 效果 | 获得途径 |
|------|------|----------|
| 火焰 | 攻击附加 2 点灼烧 (2s) | 强化房/宝箱/商店 |
| 冰霜 | 攻击附加 30% 减速 (1s) | 同上 |
| 闪电 | 攻击 15% 概率眩晕 0.5s | 同上 |
| 毒素 | 攻击附加 1 点毒伤 (可叠 3 层) | 同上 |
| 暗影 | 攻击回复 10% 伤害量 | 同上 |
| 神圣 | 对亡灵/恶魔额外 50% 伤害 | 同上 |

### 6.2 附魔与套装的关系

```
附魔和套装是独立的——互不影响, 可叠加。

例: 狂风 8 件 + 火焰附魔 + 毒素附魔
  → 攻速 0.88s 高频攻击
  → 每次攻击附加灼烧+毒伤
  → 高频触发 爆燃 (Fire+Poison) 反应
  → 狂风的风刃也带火毒
```

## 7. 掉落系统 (更新版)

### 7.1 掉落来源

| 来源 | 掉落数 | 稀有度 |
|------|--------|--------|
| 战斗房全清 | 1 件 (60% 概率) | 白色保底 |
| 宝箱房 (免费) | 1 件 | 蓝色保底 |
| 宝箱房 (金箱/钥匙) | 2 件 | 金色保底 1 件 |
| 商店 | 按售价购买 | 蓝色保底 |
| 精英战斗房 | 1 件 | 金色保底 |
| Boss 房 | 2 件 + 1 件橙色必掉 | 金色保底 + 1 件橙色套装部件 |
| 事件房 (稀有) | 1 件 | 金色 |

### 7.2 掉落槽位倾向

```
8 个槽位的掉落概率均等 (各 12.5%)。
每件装备的槽位从 8 个中随机选 1 个。

但: 连续掉落同一槽位的概率会动态降低
  例: 刚掉了武器 → 下次武器概率降至 6%, 其他槽位升至 13.4%
  目的: 防止"连续出 3 把武器"的糟糕体验
```
