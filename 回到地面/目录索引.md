## 关键架构决策（速查）

| 决策 | 内容 |
|------|------|
| 资源加载 | `AssetBundleService` 统一加载，禁止手动拖 SpriteFrame |
| UI 构建 | 纯代码生成（`new Node()` 坐标系），无 Prefab |
| 配置读取 | `GameConfig` + JSON 表，禁止魔术数字/硬编码 |
| 随机种子 | `Rng` / `RunRng` 可复现种子随机 |
| 事件系统 | `TypedEventBus` + `GameEvents` + `BattleClock` |
| 美术管线 | Agnes AI → RGBA PNG → fix meta → assets.json 注册 → RenderAssetService 接入 |
| 微信审核 | 零血液/骷髅/死亡/恶魔，死亡=光粒子，击中=闪白 |
