## 1. 项目概述

### 一句话概念
> 玩家操控角色深入随机生成的地牢，每层都是一个独立的房间，战斗胜利后选择强化，直到倒地或触及最深处——然后"回到地面"，重头再来。

### 设计支柱 (Design Pillars)

1. **走位决策重于手速** —— 实时自动攻击，但走位和技能时机决定胜负
2. **每次旅程都不同** —— 随机房间布局、随机怪物组合、随机强化选项，保证重玩性
3. **短平快的节奏** —— 单局 5–15 分钟，适合碎片化场景（通勤、等车、WC）
4. **垃圾时间也要有价值** —— 即使失败，广告变现通道依然开放，玩家不觉得白玩

### 目标数据

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 单局时长 | 5–15 min | 碎片化场景友好 |
| 首日留存 | > 35% | 肉鸽重玩性驱动 |
| 7日留存 | > 12% | 内容更新节奏保证 |
| 人均广告观看 | > 6次/日 | 复活/强化翻倍/宝箱开锁 |
| 主包大小 | < 3.5 MB | 4MB红线留余量 |

## 2. 核心玩法循环

### Moment-to-Moment (0–30s)

```
进入房间 → 怪物出现 → 角色自动攻击最近的敌人
    ↓
玩家控制走位（摇杆）、躲避（翻滚）、释放技能
    ↓
触发元素反应（火焰+毒素=爆燃！）
    ↓
全清 → 捡宝 → 选择下一步方向
```

- **进入房间**: 房间内的怪物/陷阱/宝箱布局瞬间展现
- **实时战斗**: 角色自动攻击，玩家操控走位和技能时机
- **房间结算**: 全清后掉落金币/道具
- **方向选择**: 走廊末端出现2-3个分支，玩家选择通往下一房间的方向

### 单局循环 (5–15min)

```
地面(开始)
  ↓
第1层(3-4个房间) → 弱怪 → 1个强化选项
  ↓
第2层(3-4个房间) → 中怪 → 1个强化选项
  ↓
第3层(Boss层)    → Boss → Boss掉落
  ↓
继续深入 或 死亡 → 结算统计 → [看广告复活] → 再次挑战
```

### 长线循环 (多局)

```
每局结束 → 获得魂石(根据深度/击杀数) 
  ↓
魂石 → 永久解锁: 新角色/新武器/新天赋  
  ↓
下一局 → 更强的初始配置 → 打得更深
  ↓
"回到地面" → 积累更多魂石
```

## 3. 战斗系统设计

### 基础属性

| 属性 | 范围 | 说明 |
|------|------|------|
| HP | 10–100 | 归零则死亡 |
| ATK | 1–20 | 基础攻击力 |
| DEF | 0–10 | 减伤 |
| SPD | 1–5 | 回合内行动顺序 |
| Move | 1–3 | 每回合可移动的格数 |
| Range | 1–3 | 攻击范围 |

### 回合流程

```
玩家回合:
  1. 移动 (0-Move格)
  2. 行动 (攻击/使用道具/技能/待机)
  3. 结束 → 轮到怪物

怪物回合:
  1. 按SPD顺序行动
  2. 寻路 → 靠近玩家 → 攻击
  3. 全部行动完毕 → 轮到玩家
```

### 攻击与伤害

```
伤害 = ATK + 骰子(1-6) - DEF(目标)
暴击触发: 骰子结果 == 6 → 伤害 × 1.5
```

### 房间类型

| 类型 | 概率 | 内容 |
|------|------|------|
| 战斗房 | 60% | 1-3只怪物，全清后掉落 |
| 宝箱房 | 15% | 免费开/看广告开高级箱 |
| 回复房 | 10% | 回复泉水/篝火，回复部分HP |
| 商店房 | 10% | 用金币买道具/强化 [看广告打折] |
| 精英房 | 5% | 精英怪物，掉率高 |

## 4. 肉鸽成长系统

### 每层强化选项

战斗房掉落金币 + 概率掉落道具。每层走廊末端进入**强化房**，出现 3 选 1：

```
[强化房出现选项]

属性强化:
  - HP +5
  - ATK +2
  - DEF +1
  - Move +1
  - SPD +1

武器附魔(随机):
  - 火焰附魔: 攻击+2, 灼烧2回合
  - 冰霜附魔: 攻击+1, 缓速1回合
  - 毒附魔: 攻击+1, 毒伤持续3回合
  - 吸血附魔: 攻击回复30%伤害

特殊能力(稀有):
  - 二段跳: 每回合可移动2次
  - 嘲讽: 强制怪物攻击你1回合
  - 闪避: 20%概率闪避攻击
  - 反击: 受击时60%概率反击
```

### 永久解锁系统 (魂石商店)

| 解锁项 | 魂石 | 效果 |
|--------|------|------|
| 初始HP+5 | 200 | 永久提升初始血量 |
| 初始ATK+1 | 150 | 永久提升初始攻击 |
| 角色: 战士 | 500 | 初始+1 DEF |
| 角色: 弓手 | 500 | 初始+1 Range |
| 角色: 刺客 | 800 | 初始+1 SPD, -2 HP |
| 天赋: 贪婪 | 300 | 金币掉落+20% |
| 天赋: 坚韧 | 300 | 每层回复+1 HP |

## 5. 广告变现设计 (IAA)

### 激励广告位

| 位置 | 触发场景 | 奖励 | 预期观看率 |
|------|----------|------|------------|
| 死亡复活 | 角色死亡时弹出 | 以50%HP在当前房间复活 | 40% |
| 宝箱解锁 | 宝箱房 | 开启额外高级宝箱 | 35% |
| 强化翻倍 | 强化选择后 | 额外获得1个强化选项 | 30% |
| 商店打折 | 商店房购物 | 商品价格减半 | 25% |
| 结算翻倍 | 死亡结算时 | 魂石获得×2 | 45% |
| 每日宝箱 | 主界面 | 随机奖励(金币/魂石) | 60% |

### 插屏广告

| 位置 | 时机 | 说明 |
|------|------|------|
| 死亡结算后 | 回到主界面瞬间 | 不在游戏进行中打断 |
| 每3局后 | 第3局回到主界面 | 控制频次，不刷屏 |

### Banner广告

主界面底部常驻，游戏进行中隐藏——避免遮挡战斗画面。

## 6. 地牢生成系统

### 房间+走廊结构

```
每层 = 一个"DAG图"
       入口
      / | \
    房 房  宝
    |  |    |
    精 商  回
      \ | /
       Boss房 → 下层层梯
```

- 每层**5-7个房间**
- 入口 → 2-3个分支 → 汇聚至Boss房
- 每层概率包含1个特殊房间(宝箱/回复/商店/精英)
- 层数越深，房间数量略增，怪物越强

### 怪物生成

| 层数 | 怪物配置 | Boss |
|------|----------|------|
| 1 | 史莱姆(HP:6,ATK:2)、蝙蝠(HP:4,ATK:3) | 巨型史莱姆(HP:15,ATK:4) |
| 2 | 骷髅(HP:10,ATK:4)、蜘蛛(HP:8,ATK:5) | 骷髅王(HP:25,ATK:7) |
| 3 | 恶魔(HP:15,ATK:7)、幽灵(HP:12,ATK:6) | 暗影领主(HP:35,ATK:10) |
| 4+ | 混合+强化版 | 随机Boss+强化 |
