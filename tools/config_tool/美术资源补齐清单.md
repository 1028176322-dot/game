> 版本: v1.0 | 日期: 2026-06-25
> 用途: 给 AI 生成工具的逐个资源规格说明

---

## 全局通用规格

```
色彩模式:      RGB (背景/地形) / RGBA (角色/怪物/UI/特效)
压缩要求:      保持视觉质量前提下尽量缩小, 单图集 < 512KB
画布对齐:      角色/怪物居中, 底部对齐网格线
命名规则:      [类型]_[区域/所属]_[名称]_[状态].[扩展名]
动画帧合并:    序列帧合并为 Sprite Sheet (单张 PNG, 水平排列各帧)
文件路径:      使用绝对路径 E:/game/回到地面/assets/resources/...

通用色调分区:
  遗忘森林: #4A7A4A 翠绿 + #8BAA7A 苔藓
  幽暗墓穴: #6A5A7A 灰紫 + #AAFFAA 鬼火
  熔岩火山: #AA3A2A 红橙 + #3A2A1A 焦黑
  冰封雪原: #7ABBDD 蓝白 + #DDFFFF 冰晶
  剧毒沼泽: #3A5A2A 墨绿 + #AA77AA 毒紫
  暗影深渊: #2A1A3A 紫黑 + #6644AA 虚空
  UI 基调:  #1A1A2E 深蓝黑背景 / #D4AF37 金色高亮 / #F0E8D8 米白文字
```
