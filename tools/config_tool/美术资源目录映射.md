## 二、文件命名规则

```
统一规则: [类型]_[区域/所属]_[名称]_[状态].[扩展名]

类型前缀:
  bg_       → 背景图               例: bg_room_combat_forest.png
  btn_      → 按钮                 例: btn_start_default.png
  icon_     → 图标                 例: icon_item_healingPotion.png
  char_     → 角色                 例: char_warrior_attack_0001.png
  monster_  → 普通怪物             例: monster_forest_slime_idle.png
  miniboss_ → 迷你 Boss            例: miniboss_stagBeetle_idle.png
  boss_     → 终结 Boss            例: boss_forestGuardian_idle.png
  tile_     → 地形 Tile            例: tile_forest_floor.png
  fx_       → 特效                 例: fx_reaction_burn_0001.png
  panel_    → 面板                 例: panel_settings.png
  hud_      → 战斗HUD              例: hud_hpBar_fill.png
  slider_   → 滑块                 例: slider_dot.png
  map_      → 地图组件             例: map_node_unknown.png
  card_     → 卡牌                 例: card_frame_common.png
  slot_     → 槽位                 例: slot_equip_weapon.png
  rarity_   → 稀有度边框           例: rarity_legendary.png
  set_      → 套装图标             例: icon_set_tempest.png
  bgm_      → 背景音乐             例: bgm_combat.mp3
  sfx_      → 音效                 例: sfx_ui_click.wav
  font_     → 字体                 例: font_game.ttf
```

## 四、格式要求速查

| 资源类型 | 格式 | 最大尺寸 | 颜色模式 | 备注 |
|----------|------|----------|----------|------|
| UI 元素 | PNG 32位 | 750×1334px | RGB | 按钮需三态 (default/hover/pressed) |
| 角色 | PNG 序列帧 | 64×64px | RGBA | 每个动作 4~6 帧 |
| 普通怪物 | PNG 序列帧 | 48×48px | RGBA | 每个动作 4 帧 |
| 迷你 Boss | PNG 序列帧 | 64×64px | RGBA | 每个动作 6 帧 |
| 终结 Boss | PNG 序列帧 | 96×96px | RGBA | 每个动作 8 帧 |
| 地形 Tile | PNG | 32×32px | RGB | 可平铺 |
| 房间背景 | PNG | 750×500px | RGB | 非序列帧 |
| 图标 | PNG | 48×48px | RGBA | 透明背景 |
| 特效 | Sprite Sheet PNG | 80×80px | RGBA | 4~8 帧合并在 1 张图 |
| BGM | MP3 96kbps | 单文件 < 200KB | — | 30~60 秒循环 |
| SFX | WAV | 单文件 < 100KB | — | 0.5~3 秒单发 |
| 字体 | TTF | < 1MB | — | 含 3500+ 汉字 |
