# 回到地面 (BackToGround)

回合制 Roguelike 地牢探险游戏

> **引擎**: Cocos Creator 3.x + TypeScript
> **平台**: 微信小游戏 (IAA 纯广告变现)
> **状态**: Phase 0 (设计阶段) · v0.5
> **编码**: UTF-8

---

## 项目简介

一款回合制 Roguelike 地牢探险游戏。玩家操控角色深入随机生成的地牢，每层都是一个独立的房间，战斗胜利后选择强化，直到倒地或触及最深处——然后"回到地面"，重头再来。

**设计支柱**: 决策重于操作 · 每次旅程都不同 · 短平快节奏 · 垃圾时间也有价值

## 开发路线图

| 阶段 | 内容 | 周期 |
|------|------|------|
| Phase 0: 设计 | GDD 定稿 + 数值框架 + 玩法验证 | 1 周 |
| Phase 1: 核心玩法 | 回合战斗 + 房间切换 + 怪物 AI | 3 周 |
| Phase 2: 肉鸽系统 | 强化 + 永久解锁 + 多角色 | 2 周 |
| Phase 3: 内容填充 | 5 层地牢 + 10+ 怪物 + 道具池 | 2 周 |

| Phase 5: 上线准备 | 软著 + 自审 + 备案 + 提审 | 2 周 |
