## 分批建议

### 第零批：环境验证（0 API 调用）

确认管线就绪：

```powershell
# 审计分类
python tools/art_pipeline.py audit --category backgrounds

# 查看管线状态
python tools/art_pipeline.py status
```

> ⏱ 5 分钟

---

### 第一批：小批量验证（~10 API 调用）

选 3 个类型共 5-8 个资源，用新 prompt 生成，验证质量。

**推荐的选择**（视觉可对比、API 消耗小）：

| 资源 | 理由 |
|---|---|
| `backgrounds/bg_combat_forest.jpg` | 背景清晰可辨，验证 FULLSCREEN 合同效果 |
| `ui/splash/splash_bg.jpg` | UI 全屏背景，验证一体化效果 |
| `ui/create/btn_create_confirm.png` | UI 按钮，验证无文字、风格统一 |
| `ui/character/character_card_default.png` | UI 卡片，验证 9-slice 可拉伸 |
| `ui/area/area_badge_forest.png` | 小图标，验证清晰度 |
| `icons/icon_item_key.png` | 图标，验证中心物体清晰 |

**执行**：

```powershell
python tools/art_pipeline.py generate --resource backgrounds/bg_combat_forest.jpg --self-check
python tools/art_pipeline.py generate --resource ui/splash/splash_bg.jpg --self-check
# ... 逐个
```

**验收标准**：
1. 画面更清晰（对比旧图）
2. 没有文字/伪文字
3. 没有违规元素
4. 风格与现有资源一致（森林宝石 UI）
5. 体积只在 warning 范围，不触发 palette 过度压缩

> ⏱ ~15 分钟（含 API 响应时间）
> 🔴 **人工验收节点 #1**

### 关键提示

- `--resume-failed` 可以断点续传，不怕中断
- `--self-check` 自动做技术门禁检查，通过的才标记为 validated
- 如果某分类资源量大，拆成 `--limit 20` 多次执行
- 管线不会因体积自动删除文件（已确认 post_process_generated 始终返回 True）
- 950KB 的 PNG 不会报告 failed_size，只报告 `size_warning`
