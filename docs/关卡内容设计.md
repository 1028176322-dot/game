# 关卡内容设计

> **版本**: v3.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **内容**: 6 大区域, 每区域含专属怪物 + 小关卡群 + 终结 Boss

---

## 0. 总览

```
游戏世界由 6 大区域组成, 从易到难排列:

 区域1        区域2        区域3        区域4        区域5        区域6
翠绿森林 → 幽暗墓穴 → 熔岩火山 → 冰封雪原 → 剧毒沼泽 → 暗影深渊

每局游戏随机选择其中 3 个区域组成一条冒险路线:
  例: 翠绿森林 → 冰封雪原 → 暗影深渊  (森林+雪原+深渊)
  例: 幽暗墓穴 → 熔岩火山 → 剧毒沼泽  (墓穴+火山+沼泽)
  
每个区域包含:
  ① 入口 → ② 小关卡 × 4~5 个 → ③ 终结 Boss
  
  每个小关卡 = 一个地牢楼层
    ┌─────────────────────────────────┐
    │ 入口 → 战斗 × 2~3 → 特殊房 → 迷你 Boss  │
    └─────────────────────────────────┘
  
  终结 Boss = 该区域的最终挑战
```

**每局内容量**:
```
单局 = 3 个区域 × (4~5 小关卡 × 含迷你 Boss + 1 终结 Boss)
     = 12~15 个迷你 Boss + 3 个终结 Boss
     = 约 15~18 场 Boss 战
     = 游戏时长 10~15 分钟 ✅
```

## 3. 单局游戏结构

### 3.1 开局流程

```
主界面 → 选择是否使用种子 → 开始游戏
    ↓
系统从 6 个区域中随机选择 3 个:
  规则: 区域1(新手) 必出, 剩余 2 个从区域 2~6 中随机
  确保: 每局的区域组合不同, 但新手区总是前 1~2 个
    ↓
按难度排序排列 3 个区域 → 进入第一个区域
    ↓
区域 → 小关卡1 → 小关卡2 → ... → 小关卡N → 终结 Boss
    ↓
进入下一个区域 → 重复
    ↓
3 个区域的 Boss 全部击败 → 通关!
```

**示例路线**:

| 种子 | 路线 |
|------|------|
| ABC123 | 翠绿森林 → 幽暗墓穴 → 冰封雪原 |
| DEF456 | 翠绿森林 → 熔岩火山 → 暗影深渊 |
| GHI789 | 翠绿森林 → 剧毒沼泽 → 熔岩火山 |

### 3.2 单局时长

```
一个小关卡 ≈ 3~4 分钟 (5~7 房间)
一个区域 ≈ 15~20 分钟 (4~5 小关卡 + Boss)
一局游戏 ≈ 10~15 分钟 (3 个区域, 效率通关)
探索型玩家 ≈ 15~20 分钟 (每个小关卡全清)
```

## 4. 跨局解锁与重玩

### 4.1 区域解锁

```
起始: 仅翠绿森林可用
击败森林守护者 → 解锁幽暗墓穴
击败亡灵君主 → 解锁熔岩火山
击败火焰领主 → 解锁冰封雪原
击败冰霜女王 → 解锁剧毒沼泽
击败毒沼巨兽 → 解锁暗影深渊
击败深渊魔王 → 游戏通关
```

### 4.2 重玩激励

```
通关后:
  - 新游戏+模式（怪物强度 ×1.5, 魂石 ×2）
  - 每日挑战（固定种子路线, 排行榜）
  - Boss 连战（6 个 Boss 连续战斗）
```

### 4.3 魂石商店解锁

```
魂石商店可解锁:
  区域选择权: 决定下一局从哪个区域开始 (800 魂石)
  高级开局: 从区域 3 开始, 跳过前 2 个 (500 魂石)
```
