# 关卡内容设计

> **版本**: v1.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **内容**: 5 层地牢的逐层设计，包含怪物组成、房间分布、难度曲线、Boss 设计

---

## 0. 关卡设计原则

### 0.1 每层的核心体验定位

```
层1: 教学 + 引入 — "这游戏怎么玩" 
层2: 试探 — "我开始懂了，但还没完全懂"
层3: 第一个卡点 — "我需要一个好 Build"（约 50% 玩家死在这）
层4: 硬仗 — "这里不是闹着玩的"
层5: 毕业考 — "证明你的 Build 和操作"
```

### 0.2 每层的节奏设计

```
↑ 紧张度
  │                    ╱╲  层5 Boss (高压)
  │                 ╱    ╲
  │              ╱          ╲  层4 Boss
  │           ╱    ╱╲         ╲
  │        ╱              ╲     ╲
  │     ╱   ╱╲  层2 Boss      ╲   层5 强化
  │  ╱   ╱    ╲       ╱╲        ╲
  │╱   ╱          ╱        ╲       ╲
  │   层1(新手友好)   层3(第一个卡点)
  └───────────────────────────────────→ 时间轴

波峰(战斗) —— 波谷(回复/商店/强化) —— 波峰(战斗) —— 波谷 —— 波峰(Boss)

每个波谷都是玩家的"喘息空间" — 回血、购物、强化构筑
```

### 0.3 房间分布模板

每层房间序列的基础模板（具体由种子算法做偏移）：

```
层N 房间序列:
  入口 → [战斗×1~2] → [特殊房] → [战斗×1~2] → [特殊房] → 强化房 → Boss房
                         ↑可选                 ↑可选
                       (宝箱/回复/商店/事件)
```

## 1. 第 1 层 — 新手层

**核心体验**: "引导玩家学会所有基础操作"

### 1.1 房间模板

```
入口 → 战斗(1) → 战斗(2) → 宝箱 → 战斗(3) → 强化 → Boss

固定配置:
  战斗房 1: 引导移动 + 自动攻击 (怪物极弱: HP 6, ATK 1, 无技能)
  战斗房 2: 引导翻滚 + 距离感知 (2 只远程怪)
  宝箱房: 引导开箱 + 拾取 (必定掉落 1 把钥匙)
  战斗房 3: 引导技能使用 (3 只混合怪)
  强化房: 3 选 1 (必定包含"吸血光环"选项)
  Boss: 巨型史莱姆 (HP 20)
```

### 1.2 怪物组成

| 怪物 | 数量 | HP | ATK | 说明 |
|------|------|----|-----|------|
| 小型史莱姆 | 5~7(分2批) | 6 | 1 | 移动慢, 无技能, 几乎是靶子 |
| 蝙蝠 | 2~3 | 6 | 2 | 移速快但脆, 提示"优先击杀" |
| Boss: 巨型史莱姆 | 1 | 20 | 4 | 缓慢移动, 分裂召唤小史莱姆 |

### 1.3 玩家预期状态

```
进入第 1 层: HP=25, ATK=4, 无能力
打完第 1 层: HP≈15, ATK≈4, 1 个核心能力

战斗总时长: 约 2~3 分钟
Boss 战时长: 约 10~15 秒

本地存储: 玩家在第 1 层不会死 (教学保护)
```

### 1.4 Boss: 巨型史莱姆

```
行为树:
  Phase 1 (HP 100~50%):
    - 缓慢向玩家移动 (移速: 玩家 50%)
    - 每 2 秒攻击 1 次 (近战)
    - 每 5 秒召唤 1 只小史莱姆 (最多 3 只)
    
  Phase 2 (HP < 50%):
    - 移速提升至玩家 70%
    - 攻击频率提升至每 1.5 秒 1 次
    - 停止召唤

设计意图: 让新手体验"Boss 会变强"的感觉, 但不会太难
Boss 站桩硬打也能赢, 练习翻滚可以无伤
```

## 2. 第 2 层 — 试探层

**核心体验**: "玩家开始熟悉游戏, 引入分支选择"

### 2.1 房间模板

```
入口 → 战斗 → [回复/宝箱] → 战斗 → [商店/事件] → 战斗 → 强化 → Boss
                  ↓选择                ↓选择
```

### 2.2 怪物组成

| 怪物 | 数量 | HP | ATK | 技能 |
|------|------|----|-----|------|
| 骷髅(近战) | 3~4 | 14 | 4 | 冲锋(快速接近) |
| 蜘蛛(远程) | 2~3 | 12 | 5 | 毒雾(范围持续伤) |
| 石像鬼 | 1~2 | 18 | 3 | 防御姿态(DEF+2) |
| Boss: 石像鬼统领 | 1 | 30 | 7 | 防御+反伤 |

### 2.3 玩家预期状态

```
进入第 2 层: HP≈15~20, ATK≈4~6, 1~2 个强化
打完第 2 层: HP≈10~15, ATK≈6~8, 2 个强化

通过率: 约 75% (比第 1 层明显难, 但大部分玩家能过)
```

### 2.4 Boss: 石像鬼统领

```
行为树:
  Phase 1 (HP 100~50%):
    - 架盾前进 (移速慢, DEF+4)
    - 每 2 秒挥击一次 (近战)
    - 需要绕后攻击 (背面无盾)
    
  Phase 2 (HP < 50%):
    - 收盾, 移速提升 50%
    - 攻击附带 30% 吸血
    - 偶尔使用冲刺 (快速接近)

设计意图: 引入"策略"概念 — 不是站桩打, 需要绕后
玩家需要在 Boss 面前通过走位绕到背后输出
```

## 3. 第 3 层 — 第一个卡点

**核心体验**: "Build 开始有影响, 选错流派可能死"

### 3.1 房间模板

```
入口 → 战斗(精英混合) → 宝箱/商店 → 战斗(扎堆) → [回复/事件]
                                                      ↓
                                          战斗(密集) → 强化 → Boss
```

### 3.2 怪物组成

| 怪物 | 数量 | HP | ATK | 技能 |
|------|------|----|-----|------|
| 恶魔(近战高攻) | 3~4 | 18 | 7 | 冲锋+暴击 (高威胁) |
| 幽灵(穿墙) | 2~3 | 14 | 5 | 穿墙, 物理伤害-50% |
| 恶魔召唤师(精英) | 1 | 24 | 6 | 召唤小恶魔 (上限3只) |
| Boss: 暗影猎手 | 1 | 40 | 9 | 隐身+背刺 |

### 3.3 玩家预期状态

```
进入第 3 层: HP≈20~25, ATK≈6~8, 2 个强化
打完第 3 层: HP≈5~15, ATK≈8~10, 3 个强化

通过率: 约 50% (第一个真正卡点)
关键原因:
  - 没选吸血/回复能力 → 续航不足, 被消耗死
  - 恶魔的冲锋+暴击对不翻滚的玩家是致命的
  - Boss 的隐身+背刺让走位差的玩家连续吃伤害
```

### 3.4 Boss: 暗影猎手

```
行为树:
  Phase 1 (HP 100~60%):
    - 保持距离, 远程投掷
    - 每 5 秒隐身 2 秒 → 移动到玩家背后 → 背刺 (高伤害)
    - 提示: 翻滚可以在背刺瞬间躲避
    
  Phase 2 (HP 60~30%):
    - 隐身频率提升 (每 3 秒)
    - 背刺伤害 ×1.5
    - 增加 2 只幽灵助战
    
  Phase 3 (HP < 30%):
    - 不再隐身, 转为狂暴
    - 攻速翻倍, 近战连续攻击
    - 受击概率触发瞬移

设计意图: 考验走位和翻滚时机, 不会躲背刺=死
Boss 行为模式:
  隐身 → 瞄准玩家 → 背刺 → 现形
  玩家: 看到隐身 → 开始走位 → 听到音效 → 翻滚
```

## 4. 第 4 层 — 硬仗层

**核心体验**: "精英怪物组合 + 房间变异开始出现"

### 4.1 房间模板

```
入口 → 战斗(精英) → [回复/事件] → 战斗(混合精英) → 战斗(扎堆)
                                                     ↓
                                           [商店/宝箱] → 强化 → Boss
                                                     
每层固定:
  至少 1 个精英战斗房 (精英怪替代普通怪)
  至少 1 个房间变异激活 (M01~M12)
```

### 4.2 怪物组成

| 怪物 | 数量 | HP | ATK | 技能 |
|------|------|----|-----|------|
| 暗影恶魔(精英) | 1~2 | 32 | 11 | 冲锋+召唤+吸血 (三种料) |
| 烈焰幽灵 | 2~3 | 20 | 9 | 穿墙+火焰AOE |
| 石像鬼守卫 | 2~3 | 24 | 7 | 高 DEF+架盾反伤 |
| 自爆魔像 | 2~3 | 16 | — | 5 秒倒计时自爆 (范围2格) |
| Boss: 熔岩巨像 | 1 | 52 | 12 | 全场岩浆+抓取 |

### 4.3 时间压力

```
第 4 层的典型战斗:
  战斗房 1 (精英): ~30 秒
  战斗房 2 (混合): ~25 秒
  战斗房 3 (混合精英): ~35 秒
  Boss 战: ~25 秒
  总战斗时长: ~2 分钟
  加上探索/回复/商店: 约 3~4 分钟整层
```

### 4.4 Boss: 熔岩巨像

```
行为树:
  Phase 1 (HP 100~60%):
    - 全场地面周期性喷发岩浆 (提示圈→喷发)
    - 巨像缓慢移向玩家挥击
    - 需要走位避开岩浆圈
    
  Phase 2 (HP 60~30%):
    - 岩浆喷发频率翻倍
    - 巨像使用"抓取" - 拉玩家到身边 → 重击
    - 翻滚可挣脱抓取
    
  Phase 3 (HP < 30%):
    - 全场持续喷发 (只剩几个安全点)
    - 巨像狂暴, 攻速 ×1.5
    - 新增冲锋技

设计意图: 第 4 层=弹幕地狱
玩家需要全程走位, 没有站桩时间
考验: 边躲岩浆边输出边翻滚
```

## 5. 第 5 层 — 最终挑战

**核心体验**: "你已经走到这里了, 证明你的 Build"

### 5.1 房间模板

```
入口 → 战斗(混合精英×2) → [商店/宝箱] → 战斗(扎堆+精英)
                                           ↓
                                战斗(自爆魔像群) → [回复] → 强化 → Boss
```

### 5.2 怪物组成

| 怪物 | 数量 | HP | ATK | 说明 |
|------|------|----|-----|------|
| 精英暗影恶魔 | 2 | 38 | 13 | 冲锋+召唤, 2 只同时出现 |
| 自爆魔像群 | 4~5 | 10 | — | 全自爆, 5 秒倒计时 (走位考验)|
| 幽灵集群 | 3~4 | 18 | 10 | 穿墙+冰冻弹 |
| Boss: 深渊领主 | 1 | 64 | 15 | 全屏技能+3阶段转换 |

### 5.3 玩家预期状态

```
进入第 5 层: HP≈25~35, ATK≈10~12, 3 个强化, 装备若干
打完第 5 层: 通关! (约 5~10% 玩家)

通关回报: 
  魂石: 约 290 (未翻倍)
  解锁: "通关者" 称号 (预留, Phase 5 实现)
  结算: 显示"恭喜通关! 你征服了深渊!"
```

### 5.4 Boss: 深渊领主

```
行为树:
  Phase 1 (HP 100~65%):
    - 远程黑暗弹幕 (3 发扇形)
    - 每 0.5 秒自动攻击 (高频率)
    - 召唤小暗影 (每 3 秒 1 只, 上限 3)
    
  Phase 2 (HP 65~30%):
    - 弹幕变 5 发, 有追踪效果
    - 新增"暗影波" — 全屏击退 (需要翻滚躲避)
    - 玩家每 5 秒被标记 → 3 秒后落下暗影打击(范围提示)
    
  Phase 3 (HP < 30%):
    - 不再召唤小怪
    - 弹幕频率翻倍 (0.25秒/发)
    - 全屏随机落暗影打击 (无预兆)
    - 自身移速翻倍 → 追击玩家

设计意图: 最终 Boss 是"弹幕地狱"
  不考验 Build 强度, 考验操作极限
  能过第 5 层的玩家:
  - 有一个成型 Build (至少不错)
  - 翻滚时机掌握好
  - 能在弹幕中找空隙输出
```
