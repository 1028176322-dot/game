# 关卡内容设计

> **版本**: v2.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **内容**: 关卡类型、场景主题、Boss 池、特殊玩法全量设计

---

## 0. 设计前提

**核心问题**: "不可能只有一个场景吧？"

**回答**: 是的。以下设计包含:
- **3 个场景主题**（每个主题有完全不同的视觉+怪物+Boss）
- **9 个 Boss**（每层一个，且每次随机从 Boss 池抽取）
- **4 种关卡模式**（除了主线还有每日/Boss连战/金币洞窟/深渊）
- **10+ 房间地形模板**（每种场景有自己的专属模板）

## 3. 场景轮换系统

### 3.1 单局场景流程

```
玩家进入第 1 层:
  → 从 3 个场景中随机选 1 个作为本局的主场景
  → 层1~层2 使用该场景的怪物池和 Boss 池
  → 层3~层4 使用第 2 个场景
  → 层5 使用第 3 个场景
  
  例: 本局种子=AB3K7Z
    层1-2: 遗忘森林
    层3-4: 幽暗墓穴  
    层5:   熔火深渊
    → 每次种子不同, 场景呈现顺序不同
```

### 3.2 场景解锁

```
初始: 仅可体验"遗忘森林"
      (第 1 次通关层3后解锁"幽暗墓穴")
      (第 1 次通关层5后解锁"熔火深渊")

魂石商店可解锁:
  场景选择权: 让玩家选择本局从哪个场景开始 (500 魂石)
```

## 5. 特殊玩法 (4 种)

### 5.1 每日挑战

```
解锁条件: 通关第 3 层一次
玩法: 
  - 每天固定一个种子 (基于日期生成)
  - 所有人挑战同一个地牢
  - 排行榜按层数+击杀数排序
  - 奖励: 首次通关每日挑战送 50 魂石

设计意图: 玩家之间的共同话题 —— "今天的每日挑战好难!"
```

### 5.2 Boss 连战

```
解锁条件: 通关第 5 层一次
玩法:
  - 无小怪房间, 只有 Boss
  - 连续挑战 5 个 Boss (从 9 个 Boss 池中随机选 5 个)
  - 每个 Boss 战后回复 30% HP
  - 每击败一个 Boss 获得一次强化选择
  - 击败全部 5 个 Boss 通关

设计意图: 满足"只想打 Boss"的玩家
```

### 5.3 金币洞窟

```
解锁条件: 首次到达强化房即可解锁
玩法:
  - 仅 1 层, 8~10 个战斗房
  - 无 Boss, 无强化房
  - 所有怪物金币掉落 ×3
  - 事件房 100% 给金币选项
  - 适合缺金币/魂石时刷钱

设计意图: 给玩家的"刷钱选择", 不想打主线时换个口味
```

### 5.4 深渊模式

```
解锁条件: 通关第 5 层三次
玩法:
  - 无上限层数 (从第 6 层开始, 每层加强难度)
  - 每 3 层出现一次强化房
  - 怪物属性每层 +10%
  - 层数累积到 10+ 后开始出现双 Boss
  - 死亡即结束, 不可复活 (一命模式)
  - 按到达层数结算魂石

设计意图: 给核心玩家的"挑战模式", 证明实力
```

## 7. 开发建议

### Phase 1-2 (核心玩法):
- 先只做 1 个场景 (遗忘森林) + 3 个 Boss
- 完成核心循环后再扩展

### Phase 3 (内容填充):
- 添加第 2 个场景 (幽暗墓穴) + 3 个 Boss
- 添加第 3 个场景 (熔火深渊) + 3 个 Boss

### Phase 4 (变现):
- 添加特殊玩法 (每日/Boss连战/金币/深渊)
- 特殊玩法通过广告解锁 (如每日挑战看广告免费打一次)
