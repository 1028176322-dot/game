> 任务类型: 基础设施 / 日志 ｜ 依赖: D0-0 ｜ Token Budget: 1 文件

## 输入
- `docs/2D转3D全面升级方案.md §5.4` 签名与 `LogChannel` 行为。
- `docs/ai-tasks/_architecture_report.md §6`：扫描确认无既有 `Logger`/`LogChannel`/`ILogger`（安全新建）。

## 输出
- 新增 `assets/scripts/core/Logger.ts`，导出：
  - `enum LogLevel { Debug, Info, Warn, Error }`
  - `interface LogChannel { debug(msg: string, meta?: unknown): void; info(...); warn(...); error(...); }`
  - `class Logger { channel(name: "battle"|"ai"|"scene"|"physics"|"asset"|"ui"|"audio"): LogChannel }`
  - `const ILogger = "ILogger"`（令牌，供 GameContext 注册）
  - 行为：输出格式 `[time][channel][level] msg {meta}`；每 channel 可独立设级别；Dev 默认 `Debug` 全开，Release 默认 `Error`。

- 严格按照 §5.4；通道名**固定**为 battle/ai/scene/physics/asset/ui/audio，**不得**新增通道。
- **不得**新增方法；本 Demo 只产出 Logger，**不得**改动其它文件的 `console.log`（那是后续 Demo 的职责）。
