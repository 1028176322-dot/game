## 1. 主动技能 (Active Skills)

### 1.1 设计说明

```
主动技能是玩家在自动攻击之外, 主动释放的"大招"。
它们有冷却 CD (4~10 秒), 玩家需要判断"什么时候放"。

玩家同一时间携带 2 个主动技能。
在右下角的 2 个按钮上独立显示 CD。
```

### 1.2 技能总表 (6 种)

| 技能 | CD | 效果 | 适用场景 |
|------|----|------|----------|
| **冲刺冲锋** | 5 秒 | 快速移动 3 格, 路径上敌人受 50% ATK 伤害 | 追击/逃跑/穿怪 |
| **元素爆发** | 8 秒 | 对周围 2 格所有敌人造成 150% ATK 的元素伤害 | 被包围时清场 |
| **护盾** | 6 秒 | 生成持续 2 秒的护盾, 吸收 10 点伤害 | 预判高伤害技能 |
| **治疗波** | 10 秒 | 回复 25% HP + 清除 1 个负面状态 | 残血保命 |
| **减速领域** | 7 秒 | 在当前位置生成持续 3 秒的减速场 | 风筝远程怪 |
| **锁定射击** | 4 秒 | 对当前目标造成 200% ATK 的必中伤害 | 快速斩杀残血怪 |

### 1.3 获得方式

```
角色初始: 部分角色开局自带 1 个主动技能 (如战士带护盾)
强化房 B 层: 从 3 个主动技能中选 1 个替换当前技能
遗物: "回响之珠"使治疗波 CD 减半
```

### 1.4 组合数

```
从 6 种中选 2 种携带: C(6,2) = 15 种组合
```

## 2. 核心能力 (Core Abilities)

### 2.1 设计说明

```
核心能力是玩家的"被动战斗风格"——它们改变自动攻击的行为。
选了二段斩 → 攻击变 2 下, 选了穿影 → 可以穿人。
每个核心能力从机制层面改变玩法, 而不是单纯的数值加加加。

玩家每局最多获得 3 个核心能力(每个区域 1 个强化房)。
```

### 2.2 能力总表 (12 种)

| 能力 | 自动攻击的变化 | 操作变化 |
|------|---------------|----------|
| **二段斩** | 每次攻击打 2 下 (第二下 60% 伤害) | 叠元素更快, DPS +60% |
| **穿影** | 无变化 | 可以穿越敌人和障碍物 |
| **怒吼** | 击杀怪后下次攻击 ATK×2 | 优先清小怪叠层 |
| **吸血光环** | 每次攻击回复 15% 伤害量 | 站桩更安全 |
| **弹射箭** | 攻击弹射至第 2 目标 (50% 伤害) | 群体伤害, 清场快 |
| **盾反** | 受击时 30% 概率反 100% 伤害 | 主动站到怪中间 |
| **子弹时间** | 翻滚后 2 秒内攻速翻倍 (0.5 秒/次) | 需要操作→翻滚→爆发 |
| **元素共鸣** | 触发元素反应时额外 50% AoE 伤害 | 优先带双元素 |
| **疾跑** | 移速 +30% | 风筝/走位更灵活 |
| **霜噬** | 攻击附带 30% 减速 1 秒 | 近战可黏住远程怪 |
| **浴火** | 站在荆棘上不扣血, ATK+2 | 主动站到荆棘上 |
| **圣盾** | 每 10 秒自动生成 8 点护盾 | 被动防御, 关注走位 |

### 2.3 强度分级

```
T0 (流派核心): 二段斩 / 元素共鸣 / 子弹时间 / 穿影
  → 这些能力定义了整个流派的打法

T1 (强力辅助): 吸血光环 / 盾反 / 弹射箭 / 霜噬
  → 搭配 T0 能力有 synergy

T2 (功能型): 怒吼 / 疾跑 / 浴火 / 圣盾
  → 特定场景强, 但不是流派核心
```

### 2.4 组合数

```
每局选 3 个: C(12,3) = 220 种组合
顺序影响体验: P(12,3) = 12×11×10 = 1320 种顺序
```

```
属性强化是"数值 + 取舍"——选了一个好处, 就要承受一个代价。
玩家不会无脑选——每个选择都有痛感。
```

## 4. 遗物 (Relics)

### 4.1 设计说明

```
遗物是随机获得的"惊喜"——从战斗掉落/宝箱/事件中获得。
遗物不占强化槽位, 最多同时持有 3 个。

遗物分为两种类型:
① 被动遗物 (Passive) —— 原有, 提供全局被动加成
② 主动遗物 (Active)  —— 新增, 解锁一个额外的主动技能按钮

关键区别:
  主动遗物 = 送你一个"第 3 个技能" (不占主动技能槽位)
  被动遗物 = 永久生效的全局效果

设计目标:
  一个近战狂暴战士捡到"暗影匕首" → 突然有了远程消耗手段
  一个远程狙击手捡到"寒冰护符" → 被贴脸时有了控制反制
  遗物让你突破职业限制, 创造混合玩法
```

### 4.2 操作方式

```
主动遗物获得的技能, 在屏幕 UI 上以独立的小按钮显示:

     ┌─────┐ ┌─────┐ ┌─────┐
     │ 冲刺 │ │ 护盾 │ │ 匕首 │   ← 第 3 个按钮来自遗物
     │ CD5s │ │ CD6s │ │ CD2s │      (如果有多个主动遗物,
     └─────┘ └─────┘ └─────┘       最多显示 2 个, 优先级按获得顺序)

     ↑主动技能1  ↑主动技能2  ↑遗物技能 (独立CD)
```

### 4.3 被动遗物 (原有 8 种)

| 遗物 | 效果 | 适用流派 |
|------|------|----------|
| **荆棘甲** | 受击反伤 2 点, 无 CD | 不动明王/盾反流 |
| **幸运币** | 暴击率 +10% (d6=5~6 暴击) | 任何输出流 |
| **狂战斧** | HP < 50% 时 ATK 翻倍 | 疯狗流/怒吼流 |
| **不朽石** | 每层首次死亡 1 HP 复活 + 1 秒无敌 | 所有流派 (保命)|
| **回响之珠** | 每次使用技能回复 3 HP | 技能流 |
| **暗影斗篷** | 暗区隐身效果延长 2 秒 | 穿影流 |
| **急速手套** | 自动攻击间隔从 1.0 秒 → 0.8 秒 | 所有输出流 |
| **铁甲** | 翻滚无敌帧从 0.3 秒 → 0.5 秒 | 操作型玩家 |

### 4.4 主动遗物 (新增 8 种) 

| 遗物 | CD | 效果 | 突破职业限制 | 适用场景 |
|------|----|------|-------------|----------|
| **暗影匕首** | 2 秒 | 投掷一把暗影匕首, 对目标造成 30% ATK 伤害, 可远程释放(5格) | **近战→远程消耗** | Boss 风筝, 边跑边打慢慢磨 |
| **寒冰护符** | 5 秒 | 自动冻结最近敌人 1 秒 (不可移动+不可攻击) | **近战→控场** | 被包围时控制一个, 减少压力 |
| **烈焰之环** | 3 秒 | 以自身为中心产生火焰脉冲, 范围 2 格, 造成 40% ATK 伤害 | **远程→近身自保** | 被贴脸时清场 |
| **闪烁石** | 8 秒 | 向移动方向瞬移 3 格, 路径上敌人眩晕 0.5 秒 | **无位移职业→机动** | 穿怪堆/躲技能/翻墙 |
| **引力石** | 10 秒 | 将 3 格内所有敌人拉向自己, 并造成 30% ATK 伤害 | **近战→聚怪** | 把远程怪拉到身边砍 |
| **生命链接** | 被动 | 每次受伤时, 30% 伤害量以 AoE 形式反弹给周围敌人 | **坦克→反伤AOE** | 站怪堆里受伤=全屏输出 |
| **分身符** | 12 秒 | 在当前位置生成一个持续 3 秒的分身, 吸引附近敌人攻击它 | **所有职业→逃命** | 被围时丢分身, 自己走开 |
| **时间沙漏** | 6 秒 | 减少所有技能 (含主动技能+遗物技能) CD 1 秒 | **所有职业→技能加速** | 配合短 CD 遗物形成永动机 |

### 4.5 主动遗物的战术价值

```
每个主动遗物都创造了一种"混合玩法"：

例 1: 战士(近战) 拿到 暗影匕首 (CD 2 秒远程攻击)
  → 遇到 Boss 可以保持距离, 每 2 秒扔一刀
  → 不用近身肉搏, 慢慢磨死 Boss
  → 创造了一个"远程消耗流战士"玩法
  → 与原有"盾反+护盾"形成"龟缩远程消耗"流派

例 2: 弓手(远程) 拿到 烈焰之环 (CD 3 秒近身脉冲)
  → 被近战怪贴脸时, 按一下清场
  → 不需要走位逃跑, 站桩输出
  → 创造了一个"近战弓手"玩法
  → 弥补了远程职业的薄弱环节

例 3: 法师(元素) 拿到 时间沙漏 (每 6 秒减 1 秒 CD)
  → 配合 CD 4 秒的锁定射击 → 实际 CD 变为 3 秒
  → 配合 CD 2 秒的暗影匕首 → 实际 CD 变为 1 秒
  → 创造了一个"技能机关枪"流派
```

### 4.6 遗物获得与稀有度

```
原有被动遗物保持不变 (8 种, 战斗掉落/宝箱/事件均可获得)

新增主动遗物 (8 种):
  稀有度: 金色 (比被动遗物更稀有)
  获得途径:
    宝箱房(金箱) — 必出 1 个主动遗物 (需钥匙)
    商店 — 偶尔售卖 (价格 20 金币)
    事件房 — 小概率出现
    精英战斗房掉落 — 低概率 (5%)
```

```
总遗物组合: 93 × 37 = 3,441 种 (去掉重叠部分后约 3,000+)
```

### 6.3 流派克制关系

```
爆发流 (疯狗/子弹风暴) 克制 → 龟缩流 (不动明王)
  (爆发流能在龟缩流攒够反伤前杀掉它)

龟缩流 (不动明王/圣骑士) 克制 → 风筝流 (狙击手/暗影刺客)
  (龟缩流不吃放风筝, 你打不动我)

风筝流 (狙击手/暗影刺客) 克制 → 爆发流 (疯狗/子弹风暴)
  (风筝流能躲开爆发, 慢慢磨死)

元素流 (弹幕法师/冰火行者) ↔ 均衡 (不克制也不被克制)
  (元素流的伤害类型多样, 没有绝对弱点)
```
