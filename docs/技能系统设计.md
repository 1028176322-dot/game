> **版本**: v1.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **内容**: 四层技能体系 + 每层详解 + 组合空间 + 流派构筑

## 0. 技能体系总览

```
回到地面的"技能" = 4 个维度的能力组合:

┌────────────────────────────────────────────────────────┐
│                    玩家技能体系                          │
├────────────────────────────────────────────────────────┤
│                                                        │
│  ① 主动技能 (Active Skills) × 2 个槽位                  │
│     → 战斗中手动释放, 有 CD (4~10 秒)                  │
│     → 共 6 种, 选 2 种携带                              │
│                                                        │
│  ② 核心能力 (Core Abilities) × 3 个槽位 (每区域1个)    │
│     → 改变自动攻击行为的被动能力                         │
│     → 共 12 种, 每局选 3 种                              │
│                                                        │
│  ③ 属性强化 (Stat Upgrades) × 3 个槽位                  │
│     → 数值型增益, 但带有取舍                            │
│     → 共 7 种, 每局选 3 种                               │
│                                                        │
│  ④ 遗物 (Relics) × 若干 (不占强化槽)                   │
│     → 全局特效, 从战斗/宝箱中获得                        │
│     → 共 8 种, 每局可获得 0~3 种                        │
│                                                        │
│  ⑤ 角色初始能力 (Character Base Ability) × 1           │
│     → 解锁角色后开局自带 1 个核心能力                     │
│     → 共 5 个角色 (战士/弓手/刺客/法师/狂战士)           │
│                                                        │
└────────────────────────────────────────────────────────┘
```

## 1. 主动技能 (Active Skills)

### 1.1 设计说明

```
主动技能是玩家在自动攻击之外, 主动释放的"大招"。
它们有冷却 CD (4~10 秒), 玩家需要判断"什么时候放"。

玩家同一时间携带 2 个主动技能。
在右下角的 2 个按钮上独立显示 CD。
```

### 1.2 技能总表 (6 种)

| 技能 | CD | 效果 | 适用场景 |
|------|----|------|----------|
| **冲刺冲锋** | 5 秒 | 快速移动 3 格, 路径上敌人受 50% ATK 伤害 | 追击/逃跑/穿怪 |
| **元素爆发** | 8 秒 | 对周围 2 格所有敌人造成 150% ATK 的元素伤害 | 被包围时清场 |
| **护盾** | 6 秒 | 生成持续 2 秒的护盾, 吸收 10 点伤害 | 预判高伤害技能 |
| **治疗波** | 10 秒 | 回复 25% HP + 清除 1 个负面状态 | 残血保命 |
| **减速领域** | 7 秒 | 在当前位置生成持续 3 秒的减速场 | 风筝远程怪 |
| **锁定射击** | 4 秒 | 对当前目标造成 200% ATK 的必中伤害 | 快速斩杀残血怪 |

### 1.3 获得方式

```
角色初始: 部分角色开局自带 1 个主动技能 (如战士带护盾)
强化房 B 层: 从 3 个主动技能中选 1 个替换当前技能
遗物: "回响之珠"使治疗波 CD 减半
```

### 1.4 组合数

```
从 6 种中选 2 种携带: C(6,2) = 15 种组合
```

## 2. 核心能力 (Core Abilities)

### 2.1 设计说明

```
核心能力是玩家的"被动战斗风格"——它们改变自动攻击的行为。
选了二段斩 → 攻击变 2 下, 选了穿影 → 可以穿人。
每个核心能力从机制层面改变玩法, 而不是单纯的数值加加加。

玩家每局最多获得 3 个核心能力(每个区域 1 个强化房)。
```

### 2.2 能力总表 (12 种)

| 能力 | 自动攻击的变化 | 操作变化 |
|------|---------------|----------|
| **二段斩** | 每次攻击打 2 下 (第二下 60% 伤害) | 叠元素更快, DPS +60% |
| **穿影** | 无变化 | 可以穿越敌人和障碍物 |
| **怒吼** | 击杀怪后下次攻击 ATK×2 | 优先清小怪叠层 |
| **吸血光环** | 每次攻击回复 15% 伤害量 | 站桩更安全 |
| **弹射箭** | 攻击弹射至第 2 目标 (50% 伤害) | 群体伤害, 清场快 |
| **盾反** | 受击时 30% 概率反 100% 伤害 | 主动站到怪中间 |
| **子弹时间** | 翻滚后 2 秒内攻速翻倍 (0.5 秒/次) | 需要操作→翻滚→爆发 |
| **元素共鸣** | 触发元素反应时额外 50% AoE 伤害 | 优先带双元素 |
| **疾跑** | 移速 +30% | 风筝/走位更灵活 |
| **霜噬** | 攻击附带 30% 减速 1 秒 | 近战可黏住远程怪 |
| **浴火** | 站在荆棘上不扣血, ATK+2 | 主动站到荆棘上 |
| **圣盾** | 每 10 秒自动生成 8 点护盾 | 被动防御, 关注走位 |

### 2.3 强度分级

```
T0 (流派核心): 二段斩 / 元素共鸣 / 子弹时间 / 穿影
  → 这些能力定义了整个流派的打法

T1 (强力辅助): 吸血光环 / 盾反 / 弹射箭 / 霜噬
  → 搭配 T0 能力有 synergy

T2 (功能型): 怒吼 / 疾跑 / 浴火 / 圣盾
  → 特定场景强, 但不是流派核心
```

### 2.4 组合数

```
每局选 3 个: C(12,3) = 220 种组合
顺序影响体验: P(12,3) = 12×11×10 = 1320 种顺序
```

## 3. 属性强化 (Stat Upgrades)

### 3.1 设计说明

```
属性强化是"数值 + 取舍"——选了一个好处, 就要承受一个代价。
玩家不会无脑选——每个选择都有痛感。
```

### 3.2 强化总表 (7 种)

| 强化 | 正收益 | 负取舍 |
|------|--------|--------|
| 狂战士契约 | ATK +4 | 最大 HP -3 |
| 铁壁 | DEF +3 | 移速 -15% |
| 疾风 | SPD +1 (+10% 移速) | ATK -1 |
| 长臂 | Range +1 | DEF -1 |
| 生命护符 | HP +8 | 无法使用道具 |
| 贪婪指环 | 金币掉落 +30% | 受到伤害 +1 |
| 敏捷靴 | 翻滚 CD -1 秒 | ATK -1 |

### 3.3 组合数

```
每局选 3 个: C(7,3) = 35 种组合
```

## 4. 遗物 (Relics)

### 4.1 设计说明

```
遗物是随机获得的"惊喜"——它们不占强化槽位, 从战斗掉落/宝箱/事件中获得。
遗物提供额外 synergy, 但不决定流派方向。
最多同时持有 3 个遗物。
```

### 4.2 遗物总表 (8 种)

| 遗物 | 效果 | 适用流派 |
|------|------|----------|
| **荆棘甲** | 受击反伤 2 点, 无 CD | 盾反流/不动明王 |
| **幸运币** | 暴击率 +10% (d6=5~6 暴击) | 任何输出流 |
| **狂战斧** | HP < 50% 时 ATK 翻倍 | 怒吼流/疯狗流 |
| **不朽石** | 每层首次死亡 1 HP 复活 + 1 秒无敌 | 所有流派 (保命)|
| **回响之珠** | 每次使用技能回复 3 HP | 技能流 |
| **暗影斗篷** | 暗区隐身效果延长 2 秒 | 穿影流 |
| **急速手套** | 自动攻击间隔从 1.0 秒 → 0.8 秒 | 所有输出流 |
| **铁甲** | 翻滚无敌帧从 0.3 秒 → 0.5 秒 | 操作型玩家 |

### 4.3 组合数

```
每局可获得 0~3 个: ΣC(8,0~3) = 1+8+28+56 = 93 种组合
```

## 5. 角色初始能力

### 5.1 角色总表 (5 种)

| 角色 | 初始核心能力 | 初始主动技能 | 解锁魂石 |
|------|-------------|-------------|----------|
| 战士 | 盾反 | 护盾 | 500 |
| 弓手 | 弹射箭 | 锁定射击 | 500 |
| 刺客 | 穿影 | 冲刺冲锋 | 800 |
| 法师 | 元素共鸣 | 元素爆发 | 1,000 |
| 狂战士 | 怒吼 | 治疗波 | 1,200 |

### 5.2 设计意图

```
角色不等同于"职业"——它们是一个初始能力包。
开局选了战士, 但强化房可以选二段斩走输出流。
角色只决定"你从哪里开始"，不限制"你最终成为什么"。
```

## 6. 组合空间总表

### 6.1 理论组合数

| 层级 | 内容 | 每局获得数 | 组合数 |
|------|------|-----------|--------|
| 主动技能 | 6 种, 带 2 个 | 2 | C(6,2) = **15** |
| 核心能力 | 12 种, 选 3 个 | 3 | C(12,3) = **220** |
| 属性强化 | 7 种, 选 3 个 | 3 | C(7,3) = **35** |
| 遗物 | 8 种, 随机得 0~3 个 | 0~3 | ΣC(8,0~3) = **93** |
| 角色 | 5 种, 选 1 个 | 1 | **5** |
| **理论总组合** | | | **15×220×35×93×5 ≈ 5,371 万** |

### 6.2 有效流派 (已验证可玩的组合)

```
在 5,371 万种理论组合中, 约 50~100 种是"有效"的
(有 synergy, 能打通第 3 个区域)

核心流派 (10 种, 完全设计验证):
```

| 流派 | 核心能力 | 推荐技能 | 推荐遗物 | 关键 synergy |
|------|----------|---------|----------|-------------|
| **剃刀风暴** | 二段斩+吸血光环 | 元素爆发 | 急速手套+狂战斧 | 高频攻击→叠元素→大范围反应 |
| **不动明王** | 盾反+圣盾 | 护盾 | 铁甲+荆棘甲 | 站怪堆里, 你打我等于打自己 |
| **暗影刺客** | 穿影+子弹时间 | 冲刺冲锋 | 暗影斗篷+敏捷靴 | 翻滚→穿越→爆发→翻滚 |
| **狙击手** | 弹射箭+疾跑 | 锁定射击 | 急速手套+长臂 | 远程风筝, 一发穿一串 |
| **圣骑士** | 吸血光环+圣盾 | 治疗波 | 回响之珠+不朽石 | 死不了, 回血比掉血快 |
| **弹幕法师** | 元素共鸣+二段斩 | 元素爆发 | 幸运币+急速手套 | 高频触发元素反应链 |
| **疯狗流** | 怒吼+狂战士契约 | 治疗波 | 狂战斧+幸运币 | HP 越低伤害越高, 互砍 |
| **冰火行者** | 霜噬+浴火 | 减速领域 | 铁甲+荆棘甲 | 站荆棘上增伤, 敌人走不动 |
| **子弹风暴** | 子弹时间+怒吼 | 冲刺冲锋 | 敏捷靴+狂战斧 | 翻滚→杀人→翻滚→爆发 |
| **贪婪地主** | 疾跑+贪婪指环 | 冲刺冲锋 | 幸运币+急速手套 | 快速清图+大量金币→商店 |

### 6.3 流派克制关系

```
爆发流 (疯狗/子弹风暴) 克制 → 龟缩流 (不动明王)
  (爆发流能在龟缩流攒够反伤前杀掉它)

龟缩流 (不动明王/圣骑士) 克制 → 风筝流 (狙击手/暗影刺客)
  (龟缩流不吃放风筝, 你打不动我)

风筝流 (狙击手/暗影刺客) 克制 → 爆发流 (疯狗/子弹风暴)
  (风筝流能躲开爆发, 慢慢磨死)

元素流 (弹幕法师/冰火行者) ↔ 均衡 (不克制也不被克制)
  (元素流的伤害类型多样, 没有绝对弱点)
```

## 7. 平衡性保证

| 风险 | 设计保证 |
|------|----------|
| 某流派太强 | 每层难度自适应——超模流派的玩家会打更深, 但不会碾压 |
| 某流派太弱 | 通过 DDA 补偿——弱流派的玩家会获得更多利好变异 |
| 某遗物必选 | 遗物不决定流派——只是锦上添花, 没它也能玩 |
| 某属性取舍无痛 | 所有取舍都有实际影响——选了 ATK+4 真的会死更快 |
