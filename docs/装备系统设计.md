> **版本**: v1.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **内容**: 装备类型/词缀/稀有度/掉落/管理/附魔/背包

---

## 0. 装备体系总览

```
装备在回到地面中不是"核心养成线"——核心是 Build 构筑。
装备是"锦上添花", 提供额外数值和趣味性, 但不决定流派。

装备系统设计原则:
  ① 每件装备都有名字 (而不是"铁剑+3")
  ② 词缀暗示玩法方向 (看到"吸血的幻影之刃"就知道该走吸血流)
  ③ 装备不稀缺 (每局都能拿到几件), 但不关键 (没装备也能打通)
```

### 0.1 装备与其他系统的边界

```
┌──────────────────────────────────────────────────────┐
│                   玩家战斗力                          │
├──────────────────────────────────────────────────────┤
│                                                      │
│  ① 核心能力 (决定玩法) ← 最重要的 60% 强度           │
│  ② 属性强化 (数值取舍) ← 中间 25%                    │
│  ③ 装备词缀   (额外增益) ← 锦上添花的 10%            │
│  ④ 遗物       (战略选择) ← 另外 5%                   │
│                                                      │
│  重点: 捡到好装备不会"瞬间变强", 但会让体验更丰富      │
│        没装备也可以用核心能力打通第 3 层               │
│                                                      │
└──────────────────────────────────────────────────────┘
```

## 2. 词缀系统

> 词缀系统 = 装备的"灵魂"——每件装备的名字和效果由词缀决定。

### 2.1 词缀结构

```
每个可拾取的装备随机生成 0~2 个词缀:

[前缀] + [装备基底] + [后缀]

例:
  锋利的   单手剑   之火焰   → 锋利的火焰之剑 (Prefix+Base+Suffix)
  吸血              幻影    → 吸血的幻影之刃 (Prefix+Base+Suffix)
  (无)     皮甲    之守护   → 守护皮甲 (Base+Suffix)
  沉重的   双手剑  (无)     → 沉重的双手剑 (Prefix+Base)
```

### 2.2 前缀表 (12 种)

| 前缀 | 适用类型 | 效果 |
|------|----------|------|
| 锋利的 Sharp | 武器 | ATK +2 |
| 沉重的 Heavy | 武器 | ATK +3, 攻击间隔 +0.1 秒 |
| 迅捷的 Swift | 武器/护甲 | 攻击间隔 -0.05 秒 (武器) / 移速 +5% (护甲) |
| 坚固的 Sturdy | 护甲 | DEF +2 |
| 吸血 Vampiric | 武器 | 攻击回复 15% 伤害 |
| 带电的 Shocking | 武器 | 攻击附加闪电伤害 |
| 燃烧的 Blazing | 武器 | 攻击附加火焰伤害 |
| 冰封的 Icy | 武器 | 攻击附加冰霜伤害 |
| 剧毒的 Toxic | 武器 | 攻击附加毒素伤害 |
| 暗影的 Shadowy | 武器 | 攻击附加暗影伤害 |
| 神圣的 Hallowed | 武器 | 攻击附加神圣伤害 |
| 幸运的 Lucky | 通用 | 暴击率 +5% |

### 2.3 后缀表 (12 种)

| 后缀 | 适用类型 | 效果 |
|------|----------|------|
| 之火焰 of Flames | 武器 | 普攻变为火属性 |
| 之寒冬 of Winter | 武器 | 普攻变为冰属性 |
| 之雷霆 of Thunder | 武器 | 普攻变为雷属性 |

| 之狂战士 of the Berserker | 武器 | HP < 50% 时 ATK +3 |
| 之幻影 of the Phantom | 护甲 | 进入暗区后永久隐身 |
| 之守护 of Protection | 护甲 | 每 2 秒自动回复 1 HP |
| 之荆棘 of Thorns | 护甲 | 受击反弹 2 点伤害 |
| 之力量 of Power | 通用 | ATK +1, DEF +1 |

| 之回响 of Echo | 通用 | 每次使用技能时回复 1 HP |

### 2.4 稀有度与词缀数量

| 稀有度 | 颜色 | 词缀数 | 掉落概率 | 效果说明 |
|--------|------|--------|----------|----------|
| 普通 Common | 白色 | 0 词缀 | 60% | 只有基础属性, 无词缀效果 |
| 魔法 Magic | 蓝色 | 1 词缀 | 30% | 前缀或后缀 (随机 1 个) |
| 稀有 Rare | 金色 | 2 词缀 | 9% | 前缀+后缀 (各 1 个) |
| 传奇 Legendary | 橙色 | 2 词缀+特殊 | 1% | 额外特殊效果 (如"可装备 2 个元素附魔")|

### 2.5 传奇装备特殊效果

```
传奇装备的"特殊效果"从以下池中随机 1 个:

  双元素: 可同时装备 2 个元素附魔
  暴击强化: 暴击伤害从 ×1.5 提升至 ×2.0
  吸血强化: 吸血效果从 15% 提升至 25%
  双倍词缀: 前缀和后缀效果翻倍 (如 Sharp +2 → +4)
  遗物强化: 装备时, 持有遗物上限 +1 (可持有 4 个遗物)
```

## 3. 装备掉落

### 3.1 掉落来源

| 来源 | 概率 | 保底稀有度 |
|------|------|-----------|
| 战斗房全清 | 每房 1 件, 60% 概率掉落 | 白色 |
| 宝箱房 (免费) | 必掉 1 件 | 蓝色 |
| 宝箱房 (钥匙) | 必掉 2 件 | 金色保底 1 件 |
| 商店购买 | 按售价 | 蓝色保底 |
| 精英战斗房 | 必掉 1 件 | 金色保底 |
| Boss 掉落 | 必掉 2 件 | 金色保底 1 件, 2% 概率出传奇 |
| 事件房 | 低概率 | 蓝色 |

### 3.2 掉落装备类型分布

```
武器: 40% (随角色适配倾向)
  例: 当前角色是战士 → 单手剑 50% / 双手剑 30% / 其他 20%

护甲: 35% (完全随机)
  铠甲 40% / 皮甲 35% / 长袍 25%

饰品: 25%
  戒指 60% / 项链 40%
```

### 3.3 掉落时机

```
战斗房全清后, 结算界面弹出:

┌─────────────────────────┐
│   房间清空!              │
│                         │
│   金币 +4                │
│                         │
│   [获得装备]             │
│   锋利的火焰之剑         │
│   (单手剑 ATK+2)        │
│   攻击附加火焰伤害       │
│                         │
│   [拾取]  [丢弃]         │
└─────────────────────────┘
```

## 4. 背包与装备管理

### 4.1 背包容量

| 项目 | 上限 | 说明 |
|------|------|------|
| 已装备 | 3 件 | 武器+护甲+饰品 |
| 背包库存 | 6 件 | 备用装备, 随时可替换 |
| 道具 | 5 个 | 药水/钥匙/临时Buff |

### 4.2 装备替换

```
玩家可以在背包中随时替换装备:
  打开背包 → 选择一件已装备的装备 → 选择"替换" → 选择背包中的装备 → 装备交换

替换时机: 战斗中也可以替换! (无 CD)
  例: 遇到火焰 Boss, 临时换上有火抗的护甲
  例: 远程怪多时, 换上弓武器
```

### 4.3 装备丢弃

```
背包已满时, 新获得的装备弹出选择:
  [替换] — 选择一件已装备或背包中的装备丢弃, 换上新的
  [丢弃] — 直接丢弃新装备
  
  丢弃的装备 = 消失 (不能回收, 不能卖钱)
  设计意图: 减少玩家的"囤积焦虑", 旧的不去新的不来
```

## 5. 元素附魔

### 5.1 附魔来源

```
元素附魔 = 让武器附加元素的特殊效果
不是装备词缀——附魔是独立的, 从强化房或遗物获得。

来源:
  强化房 B 层: "火焰附魔" (攻击附加火属性)
  宝箱: 随机出元素附魔卷轴
  事件房: 小概率获得
  DDA 保底: 连续 3 场未触发元素反应 → 下个宝箱必出双元素附魔武器
```

### 5.2 附魔效果

| 附魔元素 | 效果 |
|----------|------|
| 火焰附魔 | 每次自动攻击附加 2 点灼烧伤害 (持续 2 秒) |
| 冰霜附魔 | 每次自动攻击附加 30% 减速 (持续 1 秒) |
| 闪电附魔 | 每次自动攻击 15% 概率眩晕 0.5 秒 |
| 毒素附魔 | 每次自动攻击附加 1 点毒伤 (可叠加 3 层) |
| 暗影附魔 | 每次自动攻击回复 10% 伤害量 |
| 神圣附魔 | 对亡灵/恶魔类怪物额外 50% 伤害 |

### 5.3 附魔与武器词缀的关系

```
武器词缀和元素附魔是两套独立的系统, 可以叠加:

武器: 锋利的火焰之剑 (词缀: Sharp + of Flames)
  这个武器自带火属性攻击 (因为后缀 of Flames)

如果再给这个武器使用一张"冰霜附魔卷轴":
  → 武器同时拥有 火+冰 两种元素!
  → 攻击时触发 蒸汽 (Fire+Frost) 反应
  → 这就是词缀+附魔的组合 synergy
```

**唯一规则**: 一个武器最多同时持有 2 个元素附魔 (含武器词缀自带元素)。

## 6. 装备平衡

### 6.1 装备强度占比

```
装备对玩家最终战斗力的贡献约 10~15%:
  武器: 提供约 30%~50% 的 ATK 基础值
  护甲: 提供约 30%~50% 的 DEF 基础值
  饰品: 提供约 5%~10% 的额外属性

即使空手 (无装备):
  ATK 基础值仍为 4 (来自角色本身)
  DEF 基础值仍为 2 (来自角色本身)
  
所以: 没装备 ≠ 不能打, 只是略弱
```

### 6.2 装备等级 vs 层数

```
装备没有等级概念——装备只在"当前局"内有效。
每局开局的装备从 3 个槽位的白装开始 (角色初始装备)。
通过战斗掉落/宝箱/商店逐步替换更好的装备。

所以: 第 1 层的白装到第 5 层也能用 (只是不那么好)
     装备的价值随层数不变 (没有"层级衰减")
```

### 6.3 装备经济价值

```
捡到不需要的装备可以:
  丢弃 (免费, 无收益)
  保留在背包 (占 1 格)
  不卖钱! (金币系统不鼓励卖装备, 避免"装备=货币")

设计意图: 玩家的每个装备决策是"换不换"而不是"卖不卖"
```
