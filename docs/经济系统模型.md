## 1. 金币经济流 (局内)

### 1.1 收支平衡表

```
一场完整 3 层游戏的预期金币流:

       层1        层2        层3       总计
收入  14~19     18~23      22~28     54~70
支出: 药水5     药水5+钥匙10 药水5+高级钥匙15  45
净剩  +9~14     +3~8       +2~8      +9~25 (带去下局)
      (可支配)  (紧张)      (精打细算)

收入明细:
  战斗房(3个): 3~5 × 3 = 9~15
  宝箱房(1个): 2~4 = 2~4
  卖道具(可选): 约 3~5
  --------------------------
  合计: 14~24

支出明细:
  药水: 5 (保底, 大部分玩家都会买)
  钥匙: 10 (可选, 推荐买)
  高级钥匙: 15 (高风险选择)
  临时强化: 8 (本场战斗, 可选)
  洗点券: 12 (应急)
```

### 1.2 商店定价策略

| 商品 | 价格 | 成本占收入比 | 购买意愿 |
|------|------|-------------|----------|
| 回复药水 | 5 | 25~35% | ★★★ 必买 |
| 钥匙 | 10 | 50~70% | ★★☆ 推荐 |
| 临时强化 | 8 | 40~55% | ★★☆ 战斗前买 |
| 高级钥匙 | 15 | 75~100% | ★☆☆ 赌徒 |
| 洗点券 | 12 | 60~85% | ★☆☆ 应急 |

**设计原则**: 药水价格定在单层收入的 25~35% —— 买了药水还能剩钱做其他事, 但不是随便花。

### 1.3 金币上限制 99

原设计 99 上限通过数值验证:

```
一层最多收入: 3 战斗房(15) + 1 宝箱房(4) + 卖道具(5) = 24
5 层全通不花钱: 24 × 5 = 120 → 超出 99
但: 玩家不可能一层都不花钱 (至少买药水)

更现实的场景: 每层花 5~15 金币, 剩 9~10
5 层后约剩 45~50 金币

→ 99 上限够用。如果玩家攒到 99 还没花, 说明商品定价太贵。
→ [HYPOTHESIS] 99 上限合理, playtest 验证
```

## 2. 魂石经济流 (跨局)

### 2.1 收入公式 (已修正)

```
魂石 = (8 + 层数 × 8) × 到达层数 + Boss击杀奖励

示例:
  第 1 层死亡: (8+1×8) × 1 = 16
  第 2 层死亡: (8+2×8) × 2 = 48
  第 3 层死亡: (8+3×8) × 3 = 96
  第 4 层死亡: (8+4×8) × 4 = 160
  第 5 层通关: (8+5×8) × 5 + 各层Boss(10×5) = 240 + 50 = 290
```

### 2.2 解锁性价比排序

```
按"对游戏体验的提升"排序:

1. 铁胃 (250) — 回复翻倍 → 新手生存提升最大
2. 初始携带道具 (400) — 开局有药水 → 提高第一层存活率
3. 贪婪 (300) — 魂石+15% → 加速后续全解锁
4. 战士 (500) — 解锁盾反, 新角色 → 新玩法
5. 弓手 (500) — 解锁弹射箭, 新玩法
6. 狂战斧 (600) — 半血两倍输出 → 高风险高回报
7. 初始 HP+5 ×3 (600) — 每项+5HP → 稳步提升
8. 刺客 (800) — 穿影, 高机动
9. 急速手套 (600) — 攻速 0.8秒 → 提升所有流派
10. 初始 ATK+1 ×3 (450) — 稳步提升输出
11. 法师 (1000) — 元素共鸣, 元素流派核心
12. 回响之珠 (600) — 技能回血 → 技能流派核心
13. 铁甲 (600) — 翻滚 0.5秒无敌 → 操作上限提升
14. 探索者 (400) — 每层多1房 → 更多资源
15. 狂战士 (1200) — 怒吼, 最贵角色
```

### 2.3 解锁曲线 (预期)

```
玩家第一次通关第 3 层后 (约 5~8 局后):
  期望魂石收益: 96 × 翻倍概率 (约 60% 会看广告) = ~150 魂石
  累计: ~600~1200 魂石
  
→ 此时可以解锁: 铁胃 + 初始道具 + 贪婪 + 1 个角色
→ 玩家解锁了 2~3 个有意义的东西 → 有"我在变强"的感觉

10 小时后:
  期望局数: ~40 局 (平均 4 层/局)
  累计: ~4000 魂石
  
→ 此时可以解锁: 所有角色 + 核心遗物 + 部分永久升级
→ 玩家已经解锁了大部分核心内容

20 小时后:
  累计: ~8000 魂石 → 接近全解锁
```

## 3. 广告收益模型

### 3.1 广告位定价

| 广告位 | 玩家感知价值 | 经济影响 |
|--------|-------------|----------|
| 觉悟复活 | "白捡一条命" | 玩家多打一层 → 多得 48~96 魂石 |
| 双倍宝箱 | "免费的额外宝箱" | 多得 2~4 金币 (约 40~80 金币价值) |
| 额外强化 | "多一个选择" | 多选 1 个能力, 提升通关率 |
| 商店打折 | "半价!" | 省 2.5~7.5 金币 |
| 魂石翻倍 | "魂石×2!" | 直接翻倍魂石收益 |
| 每日宝箱 | "每日任务" | 固定 5~15 魂石 |
| 跑马灯 | "顺便点一下" | 3 场战斗得 1 把钥匙 |

### 3.2 广告收入预期

```
次均广告观看: 6 次/日 (目标)
  
广告位分布:  觉悟(1) + 宝箱(1) + 翻倍(1) + 跑马灯(2) + 每日(1) = 6 次
  其中觉悟 + 翻倍 + 跑马灯 = 高价值广告 (eCPM 高)
  宝箱 + 每日 = 中等
  打折 = 低概率触发
  
[HYPOTHESIS] 人均 6 次/日的目标需要:
  - 玩家平均每天玩 3~5 局 (约 30~50 分钟)
  - 每局看 1~2 个广告
  - 这个节奏对 IAA 游戏来说是合理的
```
