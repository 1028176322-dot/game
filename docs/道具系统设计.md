# 道具系统 · 完整设计

> **版本**: v1.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **内容**: 消耗品/功能道具/元素卷轴/背包管理/掉落&商店定价

---

## 0. 道具系统定位

```
道具在回到地面中是"战术储备"——玩家在战斗中按需使用的一次性资源。
道具不决定Build走向, 但能在关键时刻救命或破局。

道具与装备的区别:
  装备 → 长期穿戴, 持续生效 (8 槽位)
  道具 → 一次性使用, 用完消失 (5 背包格)
  
道具与遗物的区别:
  遗物 → 获得后永久生效 (不消耗, 不可丢弃)
  道具 → 使用后消耗 (用完就没了)
```

### 核心规则

```
① 消耗品仅限关卡内击杀怪物随机掉落 (商店不卖)
② 消耗品不可带出当前关卡 (死结算后消失)
③ 外部商店 (魂石商店) 不贩售任何消耗品
④ 商店仅售: 钥匙/高级钥匙/洗点券/元素卷轴/装备
⑤ 道具池可扩展, 后续新道具沿用同套掉落机制

设计意图:
  消耗品稀缺化 → 每个掉落都有价值
  商店不卖药水 → 金币花在装备/钥匙/卷轴上
  道具不可继承 → 每局都是"从零收集"
```

### 道具设计原则

| 原则 | 说明 |
|------|------|
| **每个道具都有用** | 没有"垃圾道具", 每种道具至少在一个场景下有价值 |
| **关键时刻才用** | 道具不是在战斗外就用的, 是"我快死了 -> 喝药" |
| **金币消耗出口** | 道具是金币的主要消耗途径之一 |
| **5 格限制** | 玩家不能囤积太多, 每次都是"带什么"的取舍 |

## 2. 道具使用机制

### 2.1 使用方式

```
战斗中使用:
  打开道具栏 (屏幕右侧道具按钮)
  → 选择道具 → 确认使用 → 道具效果立即触发 → 道具消失
  
  使用道具 0.3 秒前摇 (动画时间), 期间可被打断
  被打断则道具不消耗, 保留

战斗中不可用的道具 (非战斗道具):
  洗点券: 仅可在强化房使用
  地图卷轴: 仅可在非战斗中查看
```

### 2.2 道具快捷键

```
建议为常用道具设置快捷键:

右下角道具按钮区域:
  ┌────────────┬────────────┬────────────┬────────────┐
  │  技能A     │  技能B     │ 道具(快捷) │ 道具(背包)  │
  │  (CD)      │  (CD)      │ 药水  ×3   │  打开全部  │
  └────────────┴────────────┴────────────┴────────────┘

玩家可设置 1 个道具为"快捷道具", 战斗中以独立按钮快速使用。
建议: 大多数玩家会把"回复药水"设为快捷, 残血时一键喝药。
```

### 2.3 数量叠加

```
同类道具可叠加:
  回复药水: 最多叠加 5 个 (占 1 格)
  钥匙: 最多叠加 3 个 (占 1 格)
  各战斗药水: 最多叠加 3 个 (占 1 格)
  元素卷轴: 不叠加 (每种占 1 格, 最多 6 格)
  
实际背包限制: 5 个道具格 (含叠加)
  例: 回复药水×3(1格) + 钥匙×1(1格) + 火焰瓶×2(1格) + 元素卷轴·火(1格) + 洗点券(1格) = 5 格
```

## 3. 道具获得

### 3.1 掉落概率

```
每只怪物死亡时, 独立判定是否掉落道具:

普通怪: 12% 概率掉落 1 个道具 (白: 回复药水/战斗药水/火焰瓶=均等)
精英怪: 40% 概率掉落 1 个道具 (蓝: 大药水/净化药水/火焰瓶)
Boss: 必掉 1 个道具 (蓝/稀有 各 50%)

掉落道具类型由该区域怪物池决定:
  遗忘森林: 偏回复药水/净化药水
  幽暗墓穴: 偏战斗药水
  熔岩火山: 偏火焰瓶
  冰封雪原: 偏冰霜瓶
  剧毒沼泽: 偏净化药水
  暗影深渊: 全部掉落池
```

### 3.2 商店售卖的物品 (仅非消耗品)

```
商店只出售以下非消耗品 (按 `design/玩法与平衡性设计.md` 定价):

| 物品 | 价格 | 说明 |
|------|------|------|
| 钥匙 | 10 金币 | 开宝箱 |
| 高级钥匙 | 15 金币 | 开金箱 |
| 洗点券 | 12 金币 | 重置本层强化 |
| 元素卷轴 | 10 金币 | 武器附魔 (永久本局) |
| 随机装备 | 8~15 金币 | 蓝色/金色品质 |
| 回复符文 | 5 金币 | 站上去回 10 HP (一次性地板)← 仅此一个回复类商品 |

  ❌ 商店不卖: 回复药水/大药水/战斗药水/火焰瓶/冰霜瓶/净化药水
  (因为这些只能靠打怪掉落)
```

## 4. 道具与系统的交互

### 4.1 道具 + 核心能力

```
某些核心能力影响道具的使用效果:

  生命护符 (B层取舍): HP+8, 但无法使用道具
  → 选了它就不能用药水了! 高风险高回报

  回响之珠 (遗物): 每次使用技能回复 3 HP
  → 道具不触发回响之珠
```

### 4.2 道具 + 套装

```
圣光套装 2 件: 治疗效果 +20%
  → 回复药水 10 HP → 12 HP
  → 大回复药水 25 HP → 30 HP
  
狂怒套装 2 件: HP<50% 时 ATK +15%
  → 残血喝药回血 → 回血后 HP>50% → 狂怒效果消失
  → 策略: 是保持残血维持输出, 还是喝药保命?
```

### 4.3 道具 + 元素反应

```
火焰瓶、冰霜瓶等投掷道具:
  → 直接对目标范围内施加元素状态
  → 可以与玩家的自动攻击元素组合触发反应
  → 例: 武器带冰 → 扔火焰瓶 → 触发 蒸汽 (Fire+Frost)
```

## 5. 背包管理

### 5.1 背包容量

| 项目 | 上限 | 说明 |
|------|------|------|
| 已装备 | 8 件 | 武器/戒指/项链/头盔/胸甲/裤子/鞋子/护手 |
| 背包 (装备) | 12 件 | 备用装备 |
| 背包 (道具) | **5 格** | 含叠加 |

### 5.2 道具排序策略

```
背包内道具按类型自动排序:
  第 1 优先级: 回复药水 (最常用)
  第 2 优先级: 战斗药水 (战斗中常用)
  第 3 优先级: 钥匙 (功能性)
  第 4 优先级: 元素卷轴 (非战斗使用)
  第 5 优先级: 稀有道具 (复活币/洗点券)
```

### 5.3 道具丢弃

```
背包道具格已满时:
  新获得道具弹出选择: [替换 — 选一个已有的扔掉] / [丢弃新的]
  
不会出现"背包满了道具掉了"的强制损失——玩家主动选择。
```

## 6. 道具平衡

### 6.1 单局道具获取

```
单局(3 区域) 预期道具掉落:

  战斗房: 约 30~45 只怪 × 12% = 4~5 个消耗品
  精英怪: 每区域 2~3 只 × 40% = 2~3 个消耗品
  Boss: 每区域 1 只 × 100% = 3 个消耗品
  宝箱房: 约 2~3 个 (不含消耗品, 含装备/卷轴)
  
  总消耗品获得: 约 9~11 个

  消耗预期:
    回复药水: 约 3~5 瓶 (残血喝)
    战斗药水: 约 1~2 瓶 (Boss 战前喝)
    火焰瓶/冰霜瓶: 约 1~2 个 (可选使用)
  
  → 药水不太够 → 玩家需要省着用
  → 不能随便喝 → 每个药水都很珍贵
```

### 6.2 金币流向 (更新版)

```
由于商店不再售卖消耗品, 金币的主要开销变为:

  钥匙: 10 金币 (开宝箱)
  高级钥匙: 15 金币 (开金箱, 出橙色装备)
  元素卷轴: 10 金币 (武器附魔)
  洗点券: 12 金币 (重置强化)
  随机装备: 8~15 金币

→ 金币不再买药水, 但钥匙+卷轴+装备占了全部预算
→ 玩家依然钱不够花, 但纠结的点变了:
  "买钥匙开宝箱赌装备" vs "买卷轴强化武器" vs "攒钱买高级钥匙"
```

### 6.3 道具的局内限定

```
所有消耗品仅在本局内有效:
  死亡/结算时 → 背包内所有消耗品消失
  不能带出本局 → 每局都是从零收集
  
这包括:
  ✅ 回复药水 / 战斗药水 / 火焰瓶等 (消失)
  ✅ 元素卷轴 (附魔已生效到武器上 → 武器消失 → 附魔也消失)
  ❌ 不消失的: 魂石/角色解锁/天赋等 (跨局持续)
```

```
回复药水是唯一"战斗中即时回血"的手段。
回复房要走到那里, 治疗符文要站上去等。
药水 = 你在任何位置、任何时候都能回血。

→ 回复药水 = 最优先购买/保留的道具
→ 其他药水 = "有余钱才买"的第二优先级
```
