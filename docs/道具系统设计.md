> **版本**: v1.0 | **日期**: 2026-06-25 | **设计者**: GameDesigner
> **内容**: 消耗品/功能道具/元素卷轴/背包管理/掉落&商店定价

## 0. 道具系统定位

```
道具在回到地面中是"战术储备"——玩家在战斗中按需使用的一次性资源。
道具不决定Build走向, 但能在关键时刻救命或破局。

道具与装备的区别:
  装备 → 长期穿戴, 持续生效 (8 槽位)
  道具 → 一次性使用, 用完消失 (5 背包格)
  
道具与遗物的区别:
  遗物 → 获得后永久生效 (不消耗, 不可丢弃)
  道具 → 使用后消耗 (用完就没了)
```

### 道具设计原则

| 原则 | 说明 |
|------|------|
| **每个道具都有用** | 没有"垃圾道具", 每种道具至少在一个场景下有价值 |
| **关键时刻才用** | 道具不是在战斗外就用的, 是"我快死了 -> 喝药" |
| **金币消耗出口** | 道具是金币的主要消耗途径之一 |
| **5 格限制** | 玩家不能囤积太多, 每次都是"带什么"的取舍 |

## 2. 道具使用机制

### 2.1 使用方式

```
战斗中使用:
  打开道具栏 (屏幕右侧道具按钮)
  → 选择道具 → 确认使用 → 道具效果立即触发 → 道具消失
  
  使用道具 0.3 秒前摇 (动画时间), 期间可被打断
  被打断则道具不消耗, 保留

战斗中不可用的道具 (非战斗道具):
  洗点券: 仅可在强化房使用
  地图卷轴: 仅可在非战斗中查看
```

### 2.2 道具快捷键

```
建议为常用道具设置快捷键:

右下角道具按钮区域:
  ┌────────────┬────────────┬────────────┬────────────┐
  │  技能A     │  技能B     │ 道具(快捷) │ 道具(背包)  │
  │  (CD)      │  (CD)      │ 药水  ×3   │  打开全部  │
  └────────────┴────────────┴────────────┴────────────┘

玩家可设置 1 个道具为"快捷道具", 战斗中以独立按钮快速使用。
建议: 大多数玩家会把"回复药水"设为快捷, 残血时一键喝药。
```

### 2.3 数量叠加

```
同类道具可叠加:
  回复药水: 最多叠加 5 个 (占 1 格)
  钥匙: 最多叠加 3 个 (占 1 格)
  各战斗药水: 最多叠加 3 个 (占 1 格)
  元素卷轴: 不叠加 (每种占 1 格, 最多 6 格)
  
实际背包限制: 5 个道具格 (含叠加)
  例: 回复药水×3(1格) + 钥匙×1(1格) + 火焰瓶×2(1格) + 元素卷轴·火(1格) + 洗点券(1格) = 5 格
```

## 3. 道具获得

### 3.1 掉落

| 来源 | 概率 | 道具池 |
|------|------|--------|
| 战斗房全清 | 20% 概率掉 1 个 | 白: 回复药水/战斗药水/火焰瓶=均等 |
| 宝箱房 (木箱) | 必掉 1 个 | 白/蓝各 50% (药水/卷轴) |
| 宝箱房 (金箱/钥匙) | 必掉 2 个 | 蓝保底 1 个+稀有 1 个 (卷轴/大药水/洗点券) |
| 事件房 | 按事件选择 | 不定 |
| Boss 房 | 必掉 1 个 | 蓝/金 各 50% |
| DDA 保底 (商店) | 永远有 1 瓶 3 金币的回复药水 | 穷人的保底 |

### 3.2 商店定价逻辑

```
道具定价 = 效果价值 × 1.5 + 便利溢价

例: 回复药水 = 10 HP = 约 1/3 次回复房 ≈ 3~4 金币价值
     定价 = 3.5 × 1.5 + 0 = 5.25 → 取整 5 金币

战斗药水 = ATK+4 一场战斗 = 约 4~5 金币价值 (对比: 永久 ATK+1 魂石=150)
     定价 = 4.5 × 1.5 + 0 = 6.75 → 取整 7 → 实际定 8 (战斗药水有心理溢价)

元素卷轴 = 永久本局附魔 = 约 6~8 金币价值
     定价 = 7 × 1.5 - 1 (便利折扣) = 9.5 → 取整 10

复活币 = 一条命 = 约 12~15 金币价值
     定价 = 13 × 1.5 + 1 = 20.5 → 取整 20
```

## 4. 道具与系统的交互

### 4.1 道具 + 核心能力

```
某些核心能力影响道具的使用效果:

  生命护符 (B层取舍): HP+8, 但无法使用道具
  → 选了它就不能用药水了! 高风险高回报

  回响之珠 (遗物): 每次使用技能回复 3 HP
  → 道具不触发回响之珠
```

### 4.2 道具 + 套装

```
圣光套装 2 件: 治疗效果 +20%
  → 回复药水 10 HP → 12 HP
  → 大回复药水 25 HP → 30 HP
  
狂怒套装 2 件: HP<50% 时 ATK +15%
  → 残血喝药回血 → 回血后 HP>50% → 狂怒效果消失
  → 策略: 是保持残血维持输出, 还是喝药保命?
```

### 4.3 道具 + 元素反应

```
火焰瓶、冰霜瓶等投掷道具:
  → 直接对目标范围内施加元素状态
  → 可以与玩家的自动攻击元素组合触发反应
  → 例: 武器带冰 → 扔火焰瓶 → 触发 蒸汽 (Fire+Frost)
```

## 5. 背包管理

### 5.1 背包容量

| 项目 | 上限 | 说明 |
|------|------|------|
| 已装备 | 8 件 | 武器/戒指/项链/头盔/胸甲/裤子/鞋子/护手 |
| 背包 (装备) | 12 件 | 备用装备 |
| 背包 (道具) | **5 格** | 含叠加 |

### 5.2 道具排序策略

```
背包内道具按类型自动排序:
  第 1 优先级: 回复药水 (最常用)
  第 2 优先级: 战斗药水 (战斗中常用)
  第 3 优先级: 钥匙 (功能性)
  第 4 优先级: 元素卷轴 (非战斗使用)
  第 5 优先级: 稀有道具 (复活币/洗点券)
```

### 5.3 道具丢弃

```
背包道具格已满时:
  新获得道具弹出选择: [替换 — 选一个已有的扔掉] / [丢弃新的]
  
不会出现"背包满了道具掉了"的强制损失——玩家主动选择。
```

## 6. 道具平衡

### 6.1 单局道具预算

```
单局(3 区域) 预期道具获取:

  战斗掉落: 约 8~12 次 × 20% = 2~3 个药水/战斗道具
  宝箱房: 每区域 1~2 个 × 3 = 3~6 个道具
  商店购买: 玩家自行选择, 约 3~5 个
  Boss 掉落: 3 个 (每区域 1 个)
  
  总获得: 约 11~17 个道具

  消耗: 
    回复药水: 约 3~5 瓶 (残血喝)
    战斗药水: 约 2~3 瓶 (Boss 战前喝)
    钥匙: 约 2~3 把 (开宝箱)
    其他: 按需
  
  → 道具总获得 > 消耗, 玩家有"富余感"但不多
```

### 6.2 道具的"不可替代性"

```
回复药水是唯一"战斗中即时回血"的手段。
回复房要走到那里, 治疗符文要站上去等。
药水 = 你在任何位置、任何时候都能回血。

→ 回复药水 = 最优先购买/保留的道具
→ 其他药水 = "有余钱才买"的第二优先级
```
