## 状态

## 强制规则

### 一票否决红线
- 无真实血液/断肢/内脏/碎骨/尸体 → 死亡化光消散，受击闪白
- 无恐怖元素 → 使用宝石/星星/光效替代
- 无"死/杀/亡/尸/血"字 → 改为"击败/战败/汲取"
- 无"恶魔/地狱/亡灵" → 改为深渊霸主/炎魔步兵
- 无老虎机/转盘等赌博暗示
- 全部通用命名 + AI 原创美术

### 名称合规
- **可用**: 史莱姆/元素/魔像/精怪/天空/大地/火焰/击败/战败/生命/能量
- **禁止**: 骷髅/尸/鬼/死亡/恶魔/血/杀/死/亡/屠/虐/击杀/斩杀/鲜血/地狱

### 美术合规
- 卡通动物风格，Q版大头身，大眼突出
- 受击闪白+后仰，不可红色溅射
- 死亡化光消散，怪物圆润明亮
- Boss 走"萌凶"路线，图标无骷髅/血滴

### 功能合规
- 启动屏 <100KB，2 秒可跳过
- 首次启动弹窗用户协议/隐私政策
- 广告不遮挡核心操作，有网络异常 fallback
- 主包 <4MB，分包加载
