# 项目约定

## 状态
✅ 已完成 — 规范已固化，新增内容需同步遵守。

## 强制规则

### 目标平台
- **微信小游戏** — 所有设计/美术/代码均以此为目标
- 主包 < 4MB（目标 3.5MB）
- 美术风格为**卡通动物风**（高清卡通动物游戏美术，非像素块风，非暗黑风）
- 内容红线：不得出现"血/杀/死/尸/亡/骷髅/恶魔/地狱"等敏感词
- 美术红线：不得有真实血液/碎骨/尸体/恐怖元素，死亡=化光消散，受击=闪白
- 60FPS，同屏 ≤ 10 单位，Draw Call < 100，内存 < 200MB
- IAA 纯广告变现，碎片化场景 5-15 分钟，单场战斗 30-90 秒
- 所有广告位必须有网络异常 fallback

### 技术栈
- Cocos Creator 3.x + TypeScript，文件编码 UTF-8
- 远程仓库：git@github.com:1028176322-dot/game.git（main/develop/feature）

### 配置表
- 所有数值读取 `GameConfig.ts` + `assets/resources/config/*.json`，禁止代码硬编码
- 每条配置标注用途/合法范围/默认值（`$comment` 字段）
- 行键用 `小写单数`，字段用 `小驼峰`
- 环境隔离通过 `env/dev.json` 覆盖层

### 版权规避
- 使用通用术语，禁用已注册商标/版权保护的专有名词
- 避免直接复制其他游戏的技能/能力实现方式
- 参考来源标注"灵感来源"而非"设计参考"
- 与提灯与地下城的差异已书面记录

### 文件路径
- 全部使用相对于项目根目录的相对路径，禁止绝对路径
  - TS import: `'../core/Constants'` ✅
  - 配置引用: `'config/battle.json'` ✅
  - 文档互链: `../docs/战斗系统规格.md` ✅
