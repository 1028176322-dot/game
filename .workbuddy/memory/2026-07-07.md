### 关键教训
- 9-slice 横向小资源（300×92 卡片）AI 经常画成居中面板，必须 prompt 强制 "fill full canvas horizontally" 并用 1792×1024 横向生成
- 所有 prompt 禁用 "Label/text/Chinese/display/counter/stats" 等会诱导 AI 渲染文字的词
- 派生变体必须程序生成，不再单独 AI 重画：locked/selected/hover/active 从 base 调色派生
- Alpha 边缘残留用：chroma 清理 → alpha 侵蚀 1px → alpha bleed → 轻锐化
